Sobre el curso

Este curso está orientado a quienes desean iniciarse en el desarrollo web frontend, utilizando tecnologías esenciales como HTML, CSS, JavaScript y jQuery. A lo largo del programa, adquirirás una base sólida en programación con JavaScript, aprenderás los fundamentos del diseño web con HTML y CSS, y utilizarás jQuery para simplificar tareas comunes como la manipulación del DOM, el manejo de eventos y la creación de efectos interactivos.

Objetivos del curso

  • Dominar los fundamentos de JavaScript: variables, tipos de datos, funciones, estructuras de control y eventos.
  • Introducir conceptos esenciales de programación orientada a objetos con JavaScript.
  • Utilizar HTML y CSS para estructurar y dar estilo a páginas web.
  • Aprender a emplear jQuery para interactuar eficientemente con elementos HTML y CSS.
  • Aplicar jQuery para implementar efectos dinámicos que mejoren la experiencia del usuario.

¿A quién va dirigido?

  • Personas que se inician en la programación web.
  • Usuarios con conocimientos básicos de HTML y CSS.
  • Desarrolladores que buscan reforzar sus habilidades en tecnologías frontend.
