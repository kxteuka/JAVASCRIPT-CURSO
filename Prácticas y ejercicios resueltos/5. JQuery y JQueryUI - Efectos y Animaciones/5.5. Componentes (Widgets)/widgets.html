Spinner

El widget Spinner permite seleccionar un valor numérico dentro de un rango definido. Puedes incrementar o decrementar el valor usando las flechas o escribiendo directamente en el campo.

Accordion

El widget Accordion permite mostrar contenido en secciones que se pueden expandir y contraer.

Dialog

El widget Dialog permite mostrar un cuadro de diálogo customizable, donde podemos mostrar avisos o hacer formularios, por ejemplo.

Menu

El widget Menu permite crear un menú de navegación con opciones que pueden ser seleccionadas por el usuario.

Select

El widget Select permite crear un menú desplegable para seleccionar una opción de una lista predefinida.
