1.7. Matrices

Matrices

Una variable normal contiene un único valor. 

Una matriz es un objeto Array capaz de almacenar muchos valores a la vez. Estos son accesibles mediante un índice o clave: 

Representación de una matriz de 10 elementos.

Declaración de una matriz

Una matriz puede declararse vacía: 

var v = new Array();	// Sin valores.

También puede declararse con valores empleando la notación JSON:


Ejemplos: 

var v = ["a", "b", "c"];	// Vector de caracteres.
var v = [1, 10, 20];	// Vector de valores.

Acceso a los valores de la matriz


Para acceder a un valor contenido en una matriz debemos indicar el identificador de la matriz y el índice o clave con el que se almacenó el valor: 

            <identificador_vector>[ <índice> ]

Ejemplo: El siguiente código define un vector en el que se almacenan cuatro valores de diferentes tipos: 

// Declaracion del vector
var v = new Array();
// Asignacion de valores
v[0] = "Roger"; 
v[1] = 120; 
v[2] = 0.5; 
v[3] = false;
// Obtencion de valores
document.write( v[0] + "<br>"); 
document.write( v[1] + "<br>"); 
document.write( v[2] + "<br>"); 
document.write( v[3] + "<br>");

Longitud de una matriz. 


La longitud de una matriz indica la cantidad de valores que tiene almacenados y se obtiene mediante la propiedad length. 

        var vector = ['hola', 'mundo'];
        console.log(vector.length);   // muestra en la consola 2


Recorrido de vectores


Recorrer una vector consiste en obtener uno por uno todos los valores que contiene. Esta operación puede implementarse de tres maneras. 

Sea la siguiente matriz: 

var valores = ["Lunes", 
      "Martes", 
      "Miercoles", 
      "Jueves", 
      "Viernes", 
      "Sabado", 
      "Domingo"];

Recorrido con bucle for:

El siguiente código muestra por pantalla todos los valores contenidos en el vector empleando una variable indx que recorre todos sus índices del 0 al último = longitud-1. 

for( var indx = 0; indx < valores.length; indx++) {
	document.write("En posicion "+indx+"->"+valores[indx]+"<br>");
}

El resultado mostrado es:

* Este bucle sólo debe emplearse si los valores se han almacenado en la matriz con índices consecutivos. 


Recorrido con for( in )

El siguiente código emplea un bucle for especial que se ejecuta tantas veces como valores haya almacenados devolviendo el índice de cada uno:  

var indice; 
for(indice in valores) {
	document.write("valor es:"+ indice +"=>"+valores[indice]+"<br >"); 
}

Recorrido con for( of ) 

El siguiente código emplea otro tipo especial de bucle for que se ejecuta tantas veces como valores haya en la matriz devolviendo cada uno de ellos.  

var indice; 
for(valor of valores) {
	document.write("valor es:"+ valor +"<br/>"); 
}
