5.1. Transiciones

Transiciones.

JQuery dispone de funciones para generar ciertos efectos de manera gradual en un intervalo de tiempo logrando así efectos de desgradado y transiciones más vistosos. Para ello pueden emplearse las siguientes funciones: 

  • Efectos de aparición y desaparición ( $().fadeIn() y $().fadeOut() ) 
  • Efectos de cambio de tamaño ( $().slideUp() y $().slideDown() ) 
  • Efectos de transición ( $().show(), $().hide() ) 

Todas ellas pueden invocarse de tres modos:

  •  Sin indicar ningún parámetro : Provoca el efecto de manera inmediata.
  •  Con argumentos “slow”, “normal”, “fast” : Provoca el efecto de manera lenta, normal, o rápida respectivamente.
  •  Con argumento numérico : Equivalente al anterior, pero el efecto se extiende el tiempo indicado en milisegundos.


Efecto de desvanecimiento ( $().fadeIn(), $().fadeOut() )


Estas funciones permiten mostrar u ocultar los elementos de la selección modificando su opacidad. La velocidad el efecto depende del argumento dado: 


<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Pulsacion boton "OCULTAR"
            $("#btnOculto").click(function () {
                // Ocultacion lenta. 
                $("img").fadeOut("slow");
            });
            // Pulsacion boton "MOSTRAR"
            $("#btnVisible").click(function () {
                // Mostrado rápido
                $("img").fadeIn("fast");
            });
        });
    </script>
</head>
<body>
    <img src="images/smiley.png" /><br />
    <input type="button" id="btnOculto" value="OCULTAR" />
    <input type="button" id="btnVisible" value="MOSTRAR" />
</body>
</html>



Efecto de escalado ( $().slideUp(), $().slideDown() )


Estas funciones muestran u ocultan los elementos de la selección modificando su tamaño. La velocidad depende del argumento indicado: 


<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Pulsacion boton "OCULTAR"
            $("#btnOculto").click(function () {
                // Ocultacion en 500 milisegundos
                $("img").slideUp(500);
            });
            // Pulsacion boton "MOSTRAR"
            $("#btnVisible").click(function () {
                // Mostrado en 1000 segundos
                $("img").slideDown(1000);
            });
        });
    </script>
</head>
<body>
    <img src="images/smiley.png" /><br />
    <input type="button" id="btnOculto" value="OCULTAR" />
    <input type="button" id="btnVisible" value="MOSTRAR" />
</body>
</html>

(*) Para poder seleccionar los elementos ocultos y los visibles pueden emplearse los selectores “:hidden” y “:visible” respectivamente.

 

Efecto de transición ( $().show(), $().hide() )


Las funciones $().show() / $().hide() muestran y ocultan respectivamente los elementos de la selección alterando su opacidad y tamaño al mismo tiempo: 


<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Pulsacion boton "OCULTAR"
            $("#btnOculto").click(function () {
                // Ocultacion lenta. 
                $("img").hide("slow");
            });
            // Pulsacion boton "MOSTRAR"
            $("#btnVisible").click(function () {
                // Mostrado rápido
                $("img").show("fast");
            });
        });
    </script>
</head>
<body>
    <img src="images/smiley.png" /><br />
    <input type="button" id="btnOculto" value="OCULTAR" />
    <input type="button" id="btnVisible" value="MOSTRAR" />
</body>
</html>

 

Cambio de estado ( $().toggle(), $().fadeToggle(), $().slideToggle() ).

Estas funciones son semejantes a las anteriores, pero lo que hacen es mostrar u ocultar alternativamente los elementos de la selección cada vez que son invocadas, esto es; si el elemento es visible se oculta, y si está oculto; se muestra:

  • $().toggle() : Oculta o muestra alternativamente del mismo modo que las funciones $().show y $().hide.
  • $().fadeToggle() : Oculta o muestra alternativamente del mismo modo que las funciones $().fadeOut() y $().fadeIn()
  • $().slideToggle() : Oculta o muestra alternativamente del mismo modo que las funciones $().slideDown() y $().slideUp()

Ejemplo: El siguiente código muestra y oculta una imagen alternativamente cada vez que se pulsa el botón “VER/OCULTAR” en una transición de 1 segundo de duración empelando la función $().fadeToggle():  

<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $("#btnAccion").click(function () {
                // Si la imagen está oculta, la muestra. Si es visible, la oculta 
                // en un lapso de 1 segundo. 
                $("img").fadeToggle(1000); 
            });
        });
    </script>
</head>
<body>
    <img src="images/smiley.png" /><br />
    <input type="button" id="btnAccion" value="VER/OCULTAR" />
</body>
</html>
