3.2. Manejo de eventos con datos vinculados

Manejo de eventos con datos vinculados

Vinculo de datos a Eventos

La función $().on() permite registrar eventos vinculando datos para la función manejadora. Para ello se emplea la siguiente sintaxis:

            $(<selección>).on(<evento>, <objeto_vinculado>, <fn_manejadora>)

La inclusión de datos al registrar un evento es especialmente útil cuando se desea dar servicio a diferentes eventos empleando la misma función manejadora. En estos casos, puedes emplearse la información vinculada para indicar alguna modificación sobre la acción a realizar. 

Ejemplo: El siguiente código modifica el ejemplo visto en la lección anterior en la que mediante la pulsación de tres botones “ROJO”, “VERDE”  y “AZUL” se modifica el color de fondo de la página. En este caso, sin embargo; empleando la misma función manejadora: 

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title></title>
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <script>
        $().ready(function () {
            // Funcion manejadora
            function manejadora( event ) {
                // Obtencion del color asociado al objeto 'data'
                var color = event.data.color; 
                $('body').css({'background-color':color});
            }
            // Registro del evento click
            $('#btn_azul').on('click', {'color': '#0000ff'}, manejadora); 
            $('#btn_rojo').on('click', {'color': '#ff0000'}, manejadora); 
            $('#btn_verde').on('click', {'color': '#00ff00'}, manejadora); 
        });
    </script>
</head>
<body>
    <button id='btn_azul'>AZUL</button>
    <button id='btn_rojo'>ROJO</button>
    <button id='btn_verde'>VERDE</button>
</body>
</html>

Para ello empleamos la función $().on() para registrar el evento ‘click’ a cada botón incluyendo en cada caso un objeto con un atributo ‘color’   y el código hexadecimal del color correspondiente: 

      // Registro del evento click 
      $('#btn_azul').on('click', {'color': '#0000ff'}, manejadora); 
      $('#btn_rojo').on('click', {'color': '#ff0000'}, manejadora); 
      $('#btn_verde').on('click', {'color': '#00ff00'}, manejadora);

Por su parte la función manejadora puede obtener los valores vinculados mediante la propiedad ‘data’ del objeto recibido por el parámetro event. En este caso, para recuperar la propiedad color del objeto vinculado al evento se emplea: event.data.color: 

      // Funcion manejadora
      function manejadora( event ) {
             // Obtención del color asociado al objeto 'data'
             var color = event.data.color; 
             $('body').css({'background-color':color});
      }
Supresión de eventos

La función $().off() permite desvincular de el/los elementos seleccionados el evento indicado como argumento. Su sintaxis es la siguiente: 

           $(<selección>).off('<evento>')

Ejemplo: La siguiente página muestra una casilla de verificación y un botón “ACCION”. Cuando la casilla es marcada se registra el evento ‘click’ al botón para que muestre un cuadro de diálogo al ser pulsado. El mismo evento es desvinculado al desmarcar la casilla de modo que el botón deja de responder a la pulsación del usuario: 


<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title></title>
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <script>
        $().ready(function () {
            // Evento de cambio de estado de la casilla de verificacion
            $('input:checkbox').change(function(event) {
                if ( $(this).prop('checked')) {
                    // Evento vinculado a boton
                    $('button').click(function(event) {
                        alert("BOTON PULSADO!");
                    }); 
                } else { 
                    // Evento desvinculado de boton
                    $('button').off('click'); 
                }
            });
        });
    </script>
</head>
<body>
    <input type="checkbox">
    EVENTO ACTIVO: 
    <button>ACCION</button>
</body>
</html>


Registro múltiple de eventos

El método $().on(), también permite vincular múltiples eventos con la misma o distintas funciones manejadoras sobre los elementos seleccionados. Para ello se emplean las siguientes sintaxis: 

  • Registro de varios eventos empleando la misma función manejadora:

            // Registro del evento mouseenter y mouseleave para todas las capas. 
            $("div").on("mouseenter mouseleave", function () {
                …
            });
  • Registro de múltiples eventos con sus correspondientes funciones manejadoras:  

            $("div").on({
                mouseenter: function () {
                    alert("El puntero del ratón entra en la capa");
                },
                mouseleave: function () {
                    alert("El puntero del ratón sale de la capa");
                },
                click: function () {
                    alert("El puntero del ratón sale de la capa");
                }
            });
