2.4. Filtros

Filtros

Los filtros son métodos que se llaman a partir de una selección para filtrar aún más los elementos seleccionados. El método filter() permite aplicar un segundo selector a los elementos devueltos por una selección previa.

Ejemplo: El siguiente código devuelve las opciones <li> con la clase de estilo “current” presentes en las listas <ul> de la página:  

        $("ul li").filter(".current");


La selección inicial devuelve los elementos <li> contenidos en todos los elemento <ul> presentes en la página. A partir de ahí, el método filter() filtra aquellos elementos con estilo ‘current’.

Ejemplo: La siguiente página aplica un fondo rojo a todas las opciones <li> dentro de la lista <ul> que contengan un guión bajo en su contenido:  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('ul li').filter(':contains("_")').css("background-color", "#ff0000");
        });
    </script>
</head>
<body>
<ul>
    <li>A_</li>    
    <li>B</li>    
    <li>C</li>    
    <li>D</li>    
    <li>E_</li>    
    <li>F</li>    
    <li>G_</li>    
</ul>
</body>
</html>

También es posible emplear el método $().filter() indicando una función que determine con código qué elementos son filtrados. 

La función se ejecuta por cada elemento de la selección y recibe dos parámetros (indice y elemento ) representando cada elemento. Esta debe retornar un valor lógico por cada elemento indicando si pasa o no el filtro. 

Ejemplo: El siguiente código selecciona los mismos elementos que el ejemplo anterior pero empleando una función en el método filter(): 

   <script>
        $().ready(function () {
            $("ul li")
                .filter(function(indice, elemento) {
                        // retorna cierto si el texto del elemento contiene “_”
                        return $(elemento).text().indexOf("_") > -1;  
                 })
                 .css("background-color", "#ff0000");
    </script>


Las funciones se emplean cuando no hay selector que permita filtrar los elementos deseados.

Ejemplo: El siguiente código selecciona las opciones <li> dentro de la lista <ul> que contienen un valor numérico par: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $("ul li")
                .filter(function(indice, elemento) { 
                        // retorna cierto si el contenido es un valor numérico par. 
                        return Number($(elemento).text())%2==0;  
                 })
                 .css("background-color", "#ffff00");
        });
    </script>
</head>
<body>
<ul>
    <li>3</li>    
    <li>6</li>    
    <li>12</li>    
    <li>4</li>    
    <li>7</li>    
    <li>9</li>    
    <li>15</li>    
</ul>
</body>
</html>

Encadenamiento de métodos de filtrado.

Los métodos de filtrado se invocan a partir de una selección y devuelven otra selección, por lo que pueden encadenarse para que cada uno se aplique a la selección devuelta por el anterior. Esto permite refinar la selección de elementos.

Ejemplo: El siguiente código selecciona y asigna un contenido HTML al tercer párrafo <h3> presente dentro de un elemento con id=’content’ (por ejemplo una capa). 

        $("#content")
            .find("h3")
            .eq(2)
            .html("nuevo contenido");


También puede emplearse el encadenamiento para realizar múltiples selecciones a partir de una misma selección inicial sin necesidad de tener que hacer diferentes selecciones. Por tanto, se emplea el método $().end() que retorna la selección anterior permitiendo encadenar otros métodos de filtrado a partir de ella.

Ejemplo: El siguiente código asigna un fondo distinto a cada una de las capas de la página mediante una sola selección. Se emplea el método $().end() para encadenar los diferentes métodos de filtrado. 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $(function () {
            // Las 4 lineas siguientes forman parte de una única sentencia de código.
            $("div")
                .eq(0).css({"background-color":"#ff0000"}).end()  // fondo rojo a capa indx 0
                .eq(1).css({"background-color":"#00ff00"}).end()  // fondo verde a capa indx 1
                .eq(2).css({"background-color":"#0000ff"}).end()  // fondo azul a capa indx 2
        });
    </script>
</head>
<body>
    <div>Capa1</div>
    <div>Capa2</div>
    <div>Capa3</div>
</body>
</html>


Explicación: La selección $(‘div’) selecciona todas las capas, eq(0) selecciona la primera a la que se le asigna fondo rojo con css(). El método end() concatenado a continuación retorna entonces a la selección inicial de las tres capas, donde eq(1) selecciona la segunda cambiando su fondo a verde. El método end() concatenado retorna otra vez a la selección inicial de las tres capas donde eq(2) termina por seleccionar la última capa y cambiando su fondo a azul.

Filtro de comprobación $().is()

Este filtro devuelve un valor lógico (cierto/falso) indicando si algún elemento de la selección cumple el selector dado como argumento. Se emplea habitualmente en condicionales para validar si algún elemento de los seleccionados posee cierto estado.

Ejemplo: El siguiente código comprueba si alguno de los botones de opción seleccionados está marcado: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Hay algún botón radio marcado?
            if ( $(':radio').is(':checked') == false)
                // Ningún botón de opción marcado.
                alert("No se marco");  
            else 
                // SI. Se muestra el atributo 'value' del botón marcado
                alert("Se marco: " + $(':radio:checked').val());
        });
    </script>
</head>
    <form>
        <input type='radio' name='sexo' value='HOMBRE'/>Hombre
        <input type='radio' name='sexo' value='MUJER' checked/>Mujer   
    </form>
</body>
</html>
