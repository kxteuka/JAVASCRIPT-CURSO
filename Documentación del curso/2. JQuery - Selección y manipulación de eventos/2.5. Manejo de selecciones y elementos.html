2.5. Manejo de selecciones y elementos.

Manejo de selecciones

Todos los selectores y métodos de filtrado de JQuery devuelven una selección. 

>Una selección es en realidad un objeto de JQuery que contiene internamente una colección con todos los seleccionados. Dicha colección puede a su vez contener cero, uno o más objetos que representan los elementos seleccionados. 

 Para obtener el nº de elementos contenidos en la selección se emplea la propiedad length: 

var num_capas = $("div").length;   // Devuelve el nº de elementos <div> seleccionados

 
Cuando se emplea varias veces una misma selección es útil asignarla a una variable para no tener que repetir la selección: 

var $capas = $("div");   // La variable capas recibe la selección.

(*) Es habitual anteponer el signo $ delante de las variables que almacenan una selección para distinguirlas de las variables con otros tipos de valores. 

 

Ejemplo: El siguiente selecciona todas las capas presente en la página almacenando la selección en la variable $capas . Seguidamente se emplea la variable para seleccionar por separado cada una de las capaz mediante el método .eq() y aplicarles un color de fondo:  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $(function () {
            // Asignacion a $capas de la selección de todas las capas.
            var $capas = $("div");
            alert("El nº de capas presentes es : " + $capas.length); 
            $capas.eq(0).css({"background-color":"#ff0000"});        // selección 1º capa
            $capas.eq(1).css({"background-color":"#00ff00"});        // selección 2º capa
            $capas.eq(2).css({"background-color":"#0000ff"});        // selección 3º capa
        });
    </script>
</head>
<body>
    <div>Capa1</div>
    <div>Capa2</div>
    <div>Capa3</div>
</body>
</html>

Obtención de un elemento DOM de una selección


Los elementos DOM son los objetos que representan los elementos HTML presentes en la página tal como los entiende Javascript al margen del uso de JQuery. Estos objetos constan de atributos y métodos que permiten su manipulación propios de Javascript y diferentes de los que facilita el uso de JQuery.

Para obtener un elemento del DOM de una selección de JQuery se emplea el método get() indicando la posición en la selección del elemento que deseamos obtener.

Ejemplo: El siguiente código devuelve el objeto del DOM de la primera capa de la página a la que le asigna un nuevo contenido empleando su propiedad innerHTML de Javascript. Esta propiedad es específica de los elementos DOM y permite obtener y modificar el contenido HTML de un elemento:  

            // Obtencion de la primera capa  como elemento DOM y asignacion de contenido        
            // empleando método propio innerHTML()
            var objCapa = $('div').get(0); 
            objCapa.innerHTML = "<p>HOLA</p>";


Esto no debe confundirse con el uso del método de filtrado eq() de JQuery, el cual devuelve una selección que consta de un único elemento. Lo que obtenemos con el método get() es el propio elemento DOM para poder manipularlo con los métodos/propiedades de Javascript, al margen de JQuery.

Ejemplo: El siguiente código realiza la misma operación del ejemplo anterior pero empleando el método html() de JQuery:  

             // Selección de la primera capa como elemento JQuery y asignación de contenido con el método html()
            var objCapa = $('div').eq(0); 
            objCapa.html("<p>HOLA</p>");


La obtención de elementos DOM y su manipulación a través de los métodos y propiedades de Javascript es necesario cuando se desea interactuar con la página sin emplear métodos modificadores de JQuery ( val(), text(), html(), css()…., etc ), o cuando se están empleando otras librerías además de JQuery que requieren el acceso a los elementos DOM de la página para su funcionamiento.



Recorrido de elementos de una selección

Hay ocasiones en las que es preciso aplicar diferentes modificaciones a los elementos de una misma selección según características o alguna condición. En tales casos, puede hacerse una selección que englobe a todos los elementos con JQuery, y recorrerlos aplicando las distintas modificaciones a cada uno según sus características. 

Para acceder uno a uno a los elementos contenidos en una selección se emplea el método $().each(). Este método requiere como argumento una función que se ejecuta por cada elemento de la selección. 

$().each(function() {
        // código función anónima
});


En el código de la función puede accederse a cada elemento de la selección mediante la selección $(this):

Ejemplo: El siguiente código muestra una página con una serie de casillas de verificación y un botón “VALIDAR” que al ser pulsado selecciona todas las casillas mostrando el valor de cada una recorriendo la selección: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready( function() {
            $(':button').click(function() {
                // Selecciona todos las casillas marcadas 
                $(':checkbox:checked').each(function(){      
                    // la función se ejecuta por cada casilla de la selección.
                    console.log($(this).val());
                });
            });
        });
    </script>
</head>
<body>
    <form>
        <input type='checkbox' name='opciones' value='ciencias'>CIENCIAS<br/>
        <input type='checkbox' name='opciones' value='literatura'>LITERATURA<br/>
        <input type='checkbox' name='opciones' value='filosofia'>FILOSOFIA<br/>
        <input type='checkbox' name='opciones' value='matematicas'>MATEMATICAS<br/>
        <input type='button' value="VALIDAR">
    </form>
</body>
</html>


La selección ‘:checkbox:checked’ obtiene todas las casillas marcadas, y el método $().each() muestra en la consola de desarrollador el valor del atributo "value" de cada una de las casillas de verificación pulsadas. 

(*) Como alternativa, llamad al método $().val() a partir de la selección ‘ :checkbox:checked ’  sólo entregaría el valor de la primera casilla marcada.

Obtención de índice de elementos.

El índice determina la posición de un elemento respecto a otros situados en su mismo nivel ( elementos hermanos ). Para ello se emplea el método $().index() que retorna un valor numérico indicando la posición, tal que 0 se corresponde con el primer elemento. 

Este método puede emplearse de dos modos:

  • Sin ningún argumento, devuelve el índice del elemento contra el que se invoca al método dentro del elemento que lo contenga en la página (p.ej, una capa, una lista).

Ejemplo: La siguiente página presenta una lista de valores <ul>, donde cada opción <li> al ser pulsado muestra en la consola del navegador el texto que contiene y su índice con respecto al resto de opciones de la lista: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <style>
        li { cursor: pointer; }
    </style>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Por cada elemento <li> al ser pulsado...
            $("li").click(function () {
                // Muestra valor y posición de elemento <li> pulsado: $(this)
                console.log( $(this).text() + " en posicion: " + $(this).index());
            });
        });
    </script>
</head>
<body>
    <ul>
        <li>A</li>
        <li>B</li>
        <li>C</li>
    </ul>
</body>
</html>

Si la función $().index() se llama indicando un elemento como argumento, devuelve la posición de dicho elemento en la selección contra la que se invoca al método. Si el elemento indicado no está presente en la selección retorna el valor -1.

Ejemplo: El siguiente código es equivalente al anterior, pero empleando el método $().index() con un parámetro:  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <style>
        li { cursor: pointer; }
    </style>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Por cada elemento <li> al ser pulsado...
            $("li").click(function () {
                // Obtención del elemento <li> pulsado
                var $elemento = $(this)
                // Muestra índice del elemento <li> entre el resto de elementos <li>
                console.log( $elemento.text() + " en posicion: " + $('li').index($elemento));
            });
        });
    </script>
</head>
<body>
    <ul>
        <li>A</li>
        <li>B</li>
        <li>C</li>
    </ul>

</html>

Métodos modificadores

Una de las tareas básicas que permite JQuery además de la selección de los elementos de una página, es la obtención y modificación de los valores de sus atributos, así como estilos y contenidos.

JQuery define para ello serie de métodos que permiten tanto obtener como establecer el valor de los atributos y contenidos de los elementos seleccionados.  Estos métodos se llaman manipuladores y disponen de diferente sintaxis según se empleen para obtener o modificar valores:

  • Obtención --> Devuelve el valor del primer elemento de la selección si no es el único: 

            // Obtencion de contenido HTML del primer párrafo de la selección.
            var contenido = $("h1").html();

 

  • Modificación --> Asigna el valor a todos los elementos de la selección. El método retorna además la selección para encadenar más métodos de filtrado/modificación: 

            // Asignación de contenido HTML a todos los párrafos seleccionados. 
            $("h1").html("bacalao");

 

Manipulación de Elementos

Los siguientes son los métodos modificadores más comunes:

  • $().html() : Obtiene/Establece el contenido HTML de un elemento.
  • $().text() : Obtiene/Establece el conteindo de un elemento ( no código HTML ).
  • $().val() : Obtiene/Establece el valor de elementos de formulario (corresponde al valor del atributo HTML value ).
  • $().attr() : Obtiene/Establece el valor de un atributo HTML de un elemento.

 

$().html()

Este método permite obtener y modificar el contenido HTML de un elemento, como por ejemplo una capa o un párrafo.

Ejemplo: La sentencia siguiente sustituye el contenido HTML de todas las capas presentes en la página por el párrafo codificado en HTML: “<p>HOLA</p>”    

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $(function () {
            $("div").html('<p>HOLA</p>');   // Asignación de código HTML para todas las capas
        });
    </script>
</head>
<body>
    <div>Capa1</div>
    <div>Capa2</div>
    <div>Capa3</div>
</body>
</html>

El método html() también permite obtener el contenido HTML de un elemento seleccionado, pero si la selección contiene más de un elemento, sólo se retorna el contenido del primero. 

            // Muestra el contenido HTML de la primera capa de las seleccionadas. 
            alert($("div").html());


$().val()

Este método obtiene y modifica los valores de elementos de formularios tales como cajas de texto, áreas de texto, listas de selección, botones de opción, casillas de verificación… etc:

  • Obtener el valor de cajas de texto <input type=”text”> y elementos <textarea>.
  • Obtener el valor del elemento <option> seleccionado en listas <select> . En caso de listas múltiples devuelve una matriz con los valores de las opciones seleccionadas.
  • Obtener el valor del botones de opción <input type=”radio”> marcado combinándolo con el selector :checked.

Ejemplo: La siguiente página obtiene y muestra en la consola del navegador los valores indicados/seleccionados del formulario al pulsarse el botón “COMPROBAR”: 

 <!DOCTYPE html>
<html>
<head>

    <meta charset="utf-8" />

        $().ready(function () {

        <input type="checkbox" value="check1"> check1
        <input type="checkbox" value="check2"> check2

        <input type="button" value="COMPROBAR"/>

  • Asignar un contenido a todos los elementos de la selección.

Manipulación de atributos y propiedades

Los métodos $().attr() y $().prop() se emplean para obtener y modificar los atributos/propiedades de los elementos de una página.

Cuando el navegador carga una página, cada elemento HTML se convierte en un objeto formado el modelo de objetos del documento o DOM. Cada objeto DOM consta de una serie de propiedades que reflejan los valores actuales de los atributos del elemento.

La función attr() hace referencia a los atributos originales de los elementos presentes en el HTML de la página. La función prop() hace referencia a los valores actuales de dichos atributos en la memoria del navegador, los cuales pueden modificarse mediante JQuery/Javascript.

Para obtener el valor de un atributo o propiedad se invoca al método indicando como argumento su nombre. Si la selección contiene varios elementos sólo se retorna el valor del atributo o propiedad del primero de ellos. 

	valor = $().attr("nombre")
	valor = $().prop("nombre")

Para modificar el valor de una propiedad se añade un segundo argumento: 

$().prop("nombre", "valor")

Si la selección contiene varios elementos, la modificación se aplica a todos ellos.

Ejemplo: Sea el siguiente campo de texto de un formulario: 

<input type="text" id="texto" value="VIEJO" />

Si ejecutamos el siguiente código obtendremos los siguientes resultados: 

$("#texto").val("NUEVO");           // El control pasa a mostrar 'NUEVO'
alert($("#texto").attr("value"));   // Muestra valor original del atributo value: 'VIEJO'
alert($("#texto").prop("value"));   // Muestra valor actual de la propiedad value: 'NUEVO'


Ejemplo: La siguiente página muestra una imagen cuya propiedad src es mostrada en la consola del navegador al pulsarse el botón “ VER” y modificada al pulsar el botón “CAMBIAR”:

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $("#cambiar").click(function (event) {
            // Altera valor del atributo src del elemento <img>
               $("img").prop("src", "images/green.png");
            });
            $("#ver").click(function (event) {
               // Muestra valor del atributo src del element <img>
               console.log($("img").prop("src"));
            });
        });
    </script>
</head>

<body>
    <form id="datos">
        <img src="images/red.png" alt="texto" />
        <input type="button" id="ver" value="ver"/>   
        <input type="button" id="cambiar" value="cambiar" />
    </form>
</body>
</html>


El resultado mostrado sería el siguiente al pulsar el botón "CAMBIAR": 


Pueden modificarse varias propiedades a la vez indicando como argumento una matriz de pares propiedad:valor con la sintaxis de JSON:  

{"prop1":"valor1", "prop2":"valor", ….}
Ejemplo: El siguiente código de script muestra el uso del método prop() para asignar valores a los atributos src y alt de un elemento <img> al mismo tiempo: 

     <script>
        $().ready(function () {
            // Altera valor del atributo “src” del elemento <img>
            $("#cambiar").click(function (event) {
               $("img").prop(  {"src":"images/green.png", 
                                "alt":"modificado"});
            });
            // Muestra valor del atributo “src” del element <img>
            $("#ver").click(function (event) {
               console.log($("img").prop("src"));
               console.log($("img").prop("alt"));
            });
        });
    </script>

Manipulación de estilos

Manipulación de reglas de estilo ( $().css() )


Este método permite obtener el valor de reglas de estilo que afectan a los elementos seleccionados. De igual modo también permite añadir nuevas reglas a las ya existentes. 

Para obtener el valor de una regla de estilo se la indica como único argumento. Si la selección consta de varios elementos sólo se considera el primero.   

    $('h1').css('font-size'); // Devuelve el tamaño de la fuente establecido, p.ej: “24px”


Para asignar una regla de estilo o modificar su valor un segundo argumento con el valor deseado. Si la selección contiene varios elementos la regla se aplica a todos. 

    $('h1').css('font-size', '15px'); // establece el tamaño de fuente a '15px’


Ejemplo: Página web provista de una caja de texto y dos botones. El botón “ver” muestra por pantalla el valor de la regla de estilo “background-color” que determina el color de fondo de la caja de texto. El botón “cambiar” modifica su valor.   

<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <script src="Scripts/jquery-3.0.0.js"></script>
    <script>
        $(function () {
            // Modifica el valor del estilo "background-color” de la caja de texto
            $("#cambiar").click(function () {
                $("#texto").css("background-color","red");
            });
            // Muestra el valor del estilo “background-color” 
            $("#ver").click(function () {
                console.log($("#texto").css("background-color"));
        });
       });
    </script>
</head>
<body>
    <form id="datos">
        <input type="text" id="texto" />
        <input type="button" id="ver" value="ver" />
        <input type="button" id="cambiar" value="cambiar" />
    </form>
</body>
</html>


El resultado mostrado por pantalla es el siguiente: 


También es posible aplicar varias reglas de estilo a la vez indicando como argumento una matriz de pares regla de estilo y valor:

{‘estilo1’:’valor1’, ‘estilo2’:’valor2’,…}

Ejemplo: El siguiente código aplicado en el ejemplo anterior modifica al mismo tiempo las propiedades de estilo “background-color” y “color” de la caja de texto: 

            $("#cambiar").click(function (event) {
                $("#texto").css(
                    {"background-color": "red",  
                     "color":"white"}
                );
             });


En el caso de reglas de estilo con valores numéricos es posible incrementar y decrementar el valor actual con los operadores += y -=: 

        $('h1').css({
            'fontSize': '+=15px',    // suma 15px al tamaño original del elemento
            'paddingTop': '-=20px'   // resta 20px al padding superior original del elemento
        });

Funciones de JQuery

Las funciones de JQuery pueden dividirse en dos tipos según el modo de llamarlas:

  • Funciones de selección $().funcion() --> Son aquellas que se invocan a partir de una selección y se ejecutan para cada elemento presente en la selección.

  •  Funciones de utilidad ( $.funcion() ) --> Son aquellas que se invocan contra el operador $ directamente y reciben por parámetro los elementos sobre los que actúan.
Función $.isNumeric()
Devuelve un valor lógico indicando si el valor dado como argumento es numérico o no. 


Ejemplo: El siguiente código muestra una página con un formulario y un campo de entrada de tipo texto que se comprueba si es un valor numérico válido al pulsarse el botón de envío. Para ello se emplea la función de utilidad $.isNumeric(): 

<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="Scripts/jquery-3.0.0.js"></script>
    <script>
        $().ready(function () {
            // Evento de envio de datos de formulario
            $("#formulario").submit(function (event) {
                // Obtencion del valor inscrito en caja de texto
                var valor = $("form input:text").val();
                // Comprobación: Es un valor numérico?
                if (!$.isNumeric(valor)) {
                    // NO LO ES --> Error
                    alert("Valor no válido!")
                    event.preventDefault();
                }
            });
        });
    </script>
</head>
<body>
    <form id="formulario">
        <input type="text" name="valor" id="uno" />
        <input type="submit" value="Comprobar" />
    </form>
</body>
</html>

 

Función $.inArray()

Busca un valor en una matriz y devuelve la posición en la que se encuentra o -1 en caso contrario. La función requiere como argumentos el valor a buscar y la matriz donde buscarlo.

Ejemplo: La siguiente página muestra dos campos de texto donde el usuario debe introducir un nombre y una contraseña y pulsar el botón "Comprobar". En javascript se definen dos matrices usuarios y contraseñas con los nombres de usuarios y contraseñas registrados en posiciones correlativas. 

Cuando el usuario pulsa el botón "Comprobar" se verifica si el usuario indicado está presente en la matriz usuarios, y si la contraseña coincide con la presente en la misma posición de la matriz contraseñas . Si no alguna de las dos condiciones se cancela la acción submit del formulario.  

<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="Scripts/jquery-3.0.0.js"></script>
    <meta charset="utf-8" />
    <script>
        var usuarios = ["Roger", "Ivan", "Yuri"];
        var claves = ["ROG", "IVN", "YRI"]; 
        $().ready(function () {
            // Evento de envío de datos de formulario
            $("#formulario").submit(function (event) {
                // Obtencion del usuario indicado
                var usuario = $("form input[name='usuario']").val();
                // Obtención de la contraseña indicada
                var clave = $("form input[name='clave']").val();
                // Obtención de indice de usuario indicado en matriz usuarios
                var indice_usuario = $.inArray(usuario, usuarios);
                // Comprobación: Existe el usuario en la matriz?
                if (indice_usuario == -1) {
                    // Indice = -1 --> Usuario no existente en matriz usuarios
                    alert("Usuario desconocido");
                    event.preventDefault();    // Se cancela el SUBMIT
                // Comprobación: Clave indicada coincide con la del usuario
                } else if (clave != claves[indice_usuario]) {
                    // No coincide --> Clave incorrecta.
                    alert("Clave incorrecta");
                    event.preventDefault();   // Se cancela el SUBMIT
                }
            });
        });
    </script>
</head>
<body>
    <form id="formulario">
        Usuario<input type="text" name="usuario" /><br/>
        Clave<input type="text" name="clave" />
        <input type="submit" value="Comprobar" />
    </form>
</body>
</html>

Retorna la cadena indicada como argumento eliminando los espacios en blanco inútiles situados a izquierda y derecha. 

                var con_espacios = " HOLA ";
                var sin_espacios = $.trim(con_espacios);
                // Muesta "HOLA"; 
                alert(sin_espacios);

 

Función $.each( function( indice, valor ))

Devuelve uno a uno los valores almacenados en una matriz u objeto de Javascript. Esta función recibe como argumentos la matriz u objeto, y una función que es invocada para cada valor contenido. Dicha función recibe dos parámetros a su vez:

  • indice : Indica el índice o nombre del atributo.
  • valor : Indica el valor asociado

Ejemplo: El siguiente código muestra el uso de la función $.each() para obtener y mostrar todos los valores de una matriz y los atributos de un objeto. 

    <script>
        // Declaración de una matriz de elementos
        var matriz = ["foo", "bar", "baz"];
        // Declaracion de un objeto empleando JSO
        var objeto = { foo: "bar", baz: "bim" };

        $().ready(function () {
            // Muestra todos los valores de la matriz
            $.each(matriz, function( indice, valor ) {
                document.write( "element " + indice + " is " + valor + "<br />");
            });
            // Muestra todos los atributos del objeto
            $.each(objeto, function( atributo, valor ) {
                document.write( atributo + " : " + valor + "<br />" );            }); 
        });
    </script>


El resultado mostrado es el siguiente: 

element 0 is foo
element 1 is bar
element 2 is baz
foo : bar
baz : bim
