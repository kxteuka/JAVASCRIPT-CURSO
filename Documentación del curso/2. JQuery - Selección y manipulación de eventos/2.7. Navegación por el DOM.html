2.7. Navegación por el DOM

Obtención de elementos padres.

JQuery dispone de métodos que permiten navegar por los elementos que componen el modelo de documento (DOM). Para ello, debe seguirse la estructura jerárquica del documento distinguiendo entre los elementos padres, hijos y hermanos de cada elemento:

  • Elementos Padres --> Son aquellos que están por encima en la jerarquía y que contienen a un determinado elemento. 
  • Elementos Hijos --> Son aquellos que están por debajo en la jerarquía y que son contenidos por un determinado elemento. 
  • Elementos Hermanos --> Son aquellos que están en el mismo nivel de jerarquía contenidos en el mismo elemento.

Podemos distinguir los siguientes elementos dentro de la jerarquía tal como hemos comentado: 

  • Elemento padre : La imagen id=”imagen” tiene por padre la capa id=”base”.
  •  Elementos hijos : La capa id=”base” tiene como elementos hijos la imagen id=”imagen”, un párrafo, un encabezado <h1>, y otra capa.
  • Elementos hermanos : E elemento imagen id=”imagen”, el párrafo, el encabezado <h1>, y la capa id=”subcapa” son hermanos.

JQuery dispone de métodos que permite obtener los elementos padres, hijos y hermanos de un determinado elemento:

Obtención de elementos padre.

  • $().parent() : Devuelve el elemento padre de un elemento seleccionado. 
  • $().parents() : Devuelve todos los elementos padre incluyendo los indirectos (elementos padres del elemento padre). Este método acepta un selector como argumento para restringir los elementos a devolver.
  • $().parentsUntil() : Devuelve todos los elementos padres hasta uno determinado por el selector indicado como argumento sin incluirlo.

Ejemplo 1: Si incluimos la siguiente sección de código javascript, vemos como seleccionar el elemento padre de aquel con id="hijo" empleando para ello la función $().parent():
 

    <script>
        $().ready(function () {
            // Selecciona la capa padre de la capa id='hijo'
            $('#hijo').parent().addClass('seleccionado'); 
        });
    </script>


Ejemplo 2: La siguiente sección de código selecciona todas los elementos padres del elemento con id="hijo" empleando la función $().parents():  

    <script>
        $().ready(function () {
            // Selecciona todas las capas padre de la capa id='hijo'
            $('#hijo').parents().addClass('seleccionado'); 
        });
    </script>

El método parents() admite un selector como argumento para restringir los elementos padres seleccionados. Por ejemplo, el siguiente código selecciona aquellos elementos padre del elemento con id="hijo" que sean capas y tengan la hoja de estilo "capa":  

    <script>
        $().ready(function () {
            // Selecciona todas las capa con estilo 'capa' padres de la capa 'hijo' 
            $('#hijo').parents('div.capa').addClass('seleccionado'); 
        });
    </script>

Ejemplo 3: La siguiente sección de código selecciona todos los elementos padre del elemento con id= “nieto” hasta el elemento con id= “abuelo” empleando la función $().parentsUntil():  

    <script>
        $().ready(function () {
            // Selecciona todas las capas padre de la capa 'nieto' hasta la capa 'padre'
            $('#nieto').parentsUntil('#abuelo').addClass('seleccionado'); 
        });
    </script>

 


Obtención de elementos hijos.

JQuery dispone de los siguientes funciones para obtener los elementos hijos de un elemento seleccionado:

  • $().children() : Devuelve los elementos contenidos en el elemento seleccionado. 
  • $().find(<selector>) :  Devuelve todos los elementos contenidos en el elemento seleccionados ( tanto directos como indirectos ) que cumplan además con el selector indicado como argumento.  

Vamos a emplear la siguiente web como modelo para nuestros ejemplos. Contiene una capa ‘padre’ que contiene 3 párrafos “A”, “B”, y “C”, y una segunda capa ‘hijo’ que contiene los párrafos “ 1”, “2” y “3”:  


<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <style type="text/css">
        div {
            background-color: white; 
            border:5px solid gray;
        }
        p {
            background-color: pink;
        }
        .seleccionado {
            background-color: yellow; 
        }
        .parrafo {}
    </style>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
    </script>
</head>
<body>
    <div id='padre'>
        <p>A</p>
        <p class='parrafo'>B</p>
        <p>C</p>
        <div id='hijo'>
            <p>1</p>
            <p class='parrafo'>2</p>
            <p>3</p>
        </div>
    </div>
</body>
</html>

 
Ejemplo 1: El siguiente código Javascript selecciona todos los elementos hijos (directos) del elemento con id = ‘padre’ empleando la función $().children():  

    <script>
        $().ready(function () {
            // Selecciona los elementos hijos directos de la capa 'padre'
            $('#padre').children().addClass('seleccionado'); 
        });
    </script>

La función $().children() también acepta también como argumento un selector para restringir los elementos hijos que quieren obtenerse. Por ejemplo; el siguiente código selecciona únicamente los párrafos contenidos en el elemento con id =‘padre’: 

    <script>
        $().ready(function () {
            // Selecciona los elementos hijos parrafo directos de la capa 'padre'
            $('#padre').children('p').addClass('seleccionado'); 
        });
    </script>

Ejemplo 3: El siguiente código selecciona todos los elementos hijos (directos e indirectos) del elemento con id = ‘padre’ que sean párrafos empleando la función $().find().  Esto seleccionar por tanto los párrafos ‘A’, ‘B’, y ‘C’ de la capa id=‘padre’ y los párrafos ‘1’, ‘ 2’, y ‘3’ contenidos a su vez en la capa id=‘hijo’: 

    <script>
        $().ready(function () {
            // Selecciona todos los párrafos hijos (directos/indirectos) de la capa 'padre'
            $('#padre').find('p').addClass('seleccionado'); 
        });
    </script>

 

Obtención de elementos hermanos.

JQuery dispone de funciones que permiten obtener y recorrer los elementos ‘hermanos’ situados al mismo nivel de un elemento en el modelo de documento (DOM). Todas ellas se invocan a partir de la selección de un elemento y retornan la selección de sus respectivos elementos hermanos:

  • $().next() : Devuelve el siguiente elemento hermano.
  • $().prev() : Devuelve el anterior elemento hermano.
  • $().nextAll() : Devuelve todos los elementos hermanos anteriores
  • $().prevAll() : Devuelve todos los elementos hermanos previos.

Estas funciones pueden combinarse además con las funciones $().first() y $().last() que permiten obtener el primer y último elemento de una selección.

Ejemplo: La siguiente página muestra una lista de opciones y una serie de botones que permiten seleccionar diferentes elementos hermanos <li> contenidos en la lista ordenada <ol>:


Cada botón selecciona el elemento <li> indicado asignándole una clase de estilo para remarcarlo. Se emplean para ello las funciones $().next(), $().prev(), $().nextAll() y $().prevAll() vistas antes junto con otras funciones de JQuery:  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <style type="text/css">
        .sel {
            background-color: yellow; 
        }
        .nosel {
            background-color: white; 
        }
    </style>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('#btn_primero').click(function(event) {
                $('ol li').removeClass('sel');
                $('ol li').first().addClass('sel');         // Primer <li>
            });
            $('#btn_ultimo').click(function(event) {
                $('ol li').removeClass('sel');
                $('ol li').last().addClass('sel');          // Ultimo <li>
            });
            $('#btn_tercero').click(function(event) {
                $('ol li').removeClass('sel');
                $('ol li').eq(3).addClass('sel');           // Tercer <li>
            });
            $('#btn_siguiente').click(function(event) {
                $('ol li').removeClass('sel');
                $('ol li').eq(3).next().addClass('sel');    // Siguiente al tercer <li>
            });
            $('#btn_anterior').click(function(event) {
                $('ol li').removeClass('sel');
                $('ol li').eq(3).prev().addClass('sel');    // Anterior al tercer <li>
            });
            $('#btn_todos_siguientes').click(function(event) {
                $('ol li').removeClass('sel');
                $('ol li').eq(3).nextAll().addClass('sel'); // Siguientes al tercer <li>
            });
            $('#btn_todos_ultimos').click(function(event) {
                $('ol li').removeClass('sel');
                $('ol li').eq(3).prevAll().addClass('sel'); // Anteriores al tercer <li>
            });
        });
    </script>
</head>
<body>
    <ol>
        <li>A</li>
        <li>B</li>
        <li>C</li>
        <li>D</li>
        <li>E</li>
        <li>F</li>
        <li>G</li>
    </ol>
    <button id='btn_primero'>PRIMERO</button><br>
    <button id='btn_ultimo'>ULTIMO</button><br>
    <button id='btn_tercero'>TERCERA OPCION</button><br>
    <button id='btn_siguiente'>SIGUIENTE AL 3</button><br>
    <button id='btn_anterior'>ANTERIOR AL 3</button><br>
    <button id='btn_todos_siguientes'>TODOS SIGUIENTES AL 3</button><br>
    <button id='btn_todos_ultimos'>TODOS ANTERIORES AL 3</button><br>
</body>
</html>

La variable $opcion_seleccionada almacena inicialmente la referencia a la primera opción <li> asignándole la hoja de estilo ‘sel’ para destacarle. Con la pulsación de cada botón, se desmarca (eliminando la hoja de estilo "sel") el elemento actualmente marcado referenciado por la variable $opcion_seleccionada, se obtiene el siguiente elemento hermano correspondiente llamando a las funciones $(). next() y $().prev() a partir del elemento actual, y se le asigna la hoja "sel" para remarcarlo como el actualmente seleccionado.             background-color: white; 
        }
    </style>



    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        var $opcion_seleccionada; 
        $().ready(function () {
            // Seleccion inicial del primer elemento. 
            $opcion_seleccionada = $('ol li').first();
            $opcion_seleccionada.addClass('sel');

            $('#btn_siguiente').click(function(event) {

                $opcion_seleccionada = $opcion_seleccionada.next();   // Siguiente <li>

            $('#btn_anterior').click(function(event) {
