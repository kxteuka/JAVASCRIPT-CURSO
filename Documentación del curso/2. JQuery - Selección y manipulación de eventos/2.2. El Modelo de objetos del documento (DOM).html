2.2. El Modelo de objetos del documento ( DOM )

Jerarquía de Objetos de una Página Web

JQuery se emplea básicamente para la manipulación de los elementos que componen las páginas (capas, tablas, elementos de formularios, imágenes, enlaces..., etc ). Estos se organizan y son accesibles a través del modelo de objetos de documento 

El modelo de objetos de documento (DOM) es una estructura jerárquica que contiene un objeto por cada una de las etiquetas presentes en el código HTML. 

Ejemplo: Sea por tanto la siguiente página web de ejemplo:   

<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
        <title>New Web Project</title>
        <script src="codigo.js"/> 
    </head>
	<body>
	     <h1>My Header</h1>
	     <a href="page2.html">My link</a>
	</body>
</html>

El DOM de una página web permite mediante Javascript/JQuery obtener y modificar el contenido y características de los elementos de la página web tal como se muestran al usuario en su navegador. Algunas de las operaciones posibles son las siguientes: 

    • Modificar la imagen mostrada por un elemento <img>
    • Modificar el contenido de un elemento como un párrafo, o una capa añadiendo, modificando o eliminando otros elementos HTML.
    • Modificar el estilo asignado, o asignar una clase de estilo diferente a un elemento para modificar su aspecto.
    • Obtener el valor introducido en una caja de texto <input type=’text’>

Es importante tener claro que estas operaciones no tienen ningún efecto sobre el diseño o contenido de la página almacenada en el servidor. Sólo afectan a la representación de la misma en el navegador del usuario.
