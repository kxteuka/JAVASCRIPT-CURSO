2.6. Manipulación del modelo de objeto de documento

Creación y Agregado de Elementos.

Cuando una página web se carga cada etiqueta HTML se convierte en un objeto parte de una estructura jerárquica encabezada por el objeto Document como raíz. Esta estructura es el modelo de objetos de documento ( DOM ).

JQuery no sólo facilita seleccionar y modificar el contenido, atributos y estilos de los elementos presentes en el DOM. También dispone de métodos para manipular la propia estructura del documento, creando e insertando nuevos objetos, así como moviendo y eliminando los ya existentes. 

Creación de elementos.

JQuery permite crear nuevos elementos y añadirlos a la página para mostrar contenidos no presentes inicialmente. Esto puede hacerse de dos maneras:

  • Creando una selección $() con el código HTML del elemento a crear: 

// Crea un parrafo con un texto
var $parrafo = $("<p>This is a new paragraph</p>");
// Crea una opción de lista con estilo 'new'
var $opcion =  $("<li class=\"new\">new list item</li>");

 

  • Creando una selección $() indicando como primer argumento el elemento HTML que se desea crear, seguido del valor de sus correspondientes atributos en formato JSON:  

// Nuevo hipervínculo a página 'foo.html'
var $enlace = $("<a/>", {
                          html: "This is a <strong>new</strong> link",
                          "class": "new", 
                          href: "foo.html"
                        });

(*) El propiedad ‘class’ va entrecomillada debido a que coincide con la palabra clave class de Javascript a diferencia de los atributos html y href.


En ambos casos se obtiene una selección con el elemento recién creado. Para que éstos formen parte de la página deben añadirse al DOM. 


Añadido de elementos.


JQuery define diferentes funciones que permiten añadir nuevos elementos al DOM de diferentes maneras: 

Función $().append()

Añade un elemento hijo al/los elementos seleccionados. El método se llama a partir de una selección al elemento o elementos en los que se quiere añadir el nuevo elemento y recibe como argumento el elemento que se desea añadir.  

$(<elemento_al_que_se_añade>).append(<elemento_que_se_añade>)

Ejemplo: El siguiente código agrega cada vez que se pulsa el botón "ACCION", un elemento <li> con el texto “Nueva Opcion” a la lista <li>:  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            var opcion = 4; 
            $('button').click(function() {
  // Creacion y configuración del nuevo elemento <li>
                $nueva_opcion = $('<li>', {text: 'opcion' + opcion}); 
  // Agregado del nuevo elemento el último de la lista <ul>
                $('ul').append($nueva_opcion); 
                opcion++; 
            });
        });
    </script>
</head>
<body>
    <ul><li>opcion1</li>
        <li>opcion2</li>
        <li>opcion3</li></ul> 
    <button>ACCION</button>    
</body>
</html>

Función $().appendTo()

Este método hace lo mismo que el anterior pero cambiando la sintaxis. El método se llama a partir de la selección del elemento que se quiere añadir, y recibe como argumento el elemento al que se añade. 

$(<elemento_que_se_añade>).appendTo(<elemento_al_que_se_añade>)

Ejemplo: Esta es la misma página de ejemplos anteriores pero empleando appendTo(): 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            var opcion = 4; 
            $('button').click(function() {
  // Creacion y configuración del nuevo elemento <li>
                $nueva_opcion = $('<li>', {text: 'opcion' + opcion}); 
  // Agregado del nuevo elemento el último de la lista <ul>
                $nueva_opcion.appendTo("ul"); 
                opcion++; 
            });
        });
    </script>
</head>
<body>
    <ul><li>opcion1</li>
        <li>opcion2</li>
        <li>opcion3</li></ul> 
    <button>ACCION</button>    
</body>

Ambas funciones desempeñan la misma tarea exactamente salvando la diferencia de su sintaxis, por lo que el uso de una o la otra depende más de la preferencia del programador. 


Función $().prepend()

Método equivalente a $().append() pero que añade un elemento como primer hijo del elemento o elementos seleccionados. Su sintaxis es la misma que append(); se llama a partir de la selección del elemento a los que se va añadir el elemento dado como argumento.  

$(<elemento_al_que_se_añade>).prepend(<elemento_a_añadir>)

Ejemplo: La siguiente página añade elementos <li> al principio de la lista <ul> cada vez que se pulsa el botón “ACCION”: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            var opcion = 4; 
            $('button').click(function() {
  // Creacion y configuración del nuevo elemento <li>
                $nueva_opcion = $('<li>', {text: 'opcion' + opcion}); 
  // Agregado del nuevo elemento el primero de la lista <ul>
                $('ul').prepend($nueva_opcion); 
                opcion++; 
            });
        });
    </script>
</head>
<body>
    <ul>
        <li>opcion1</li>
        <li>opcion2</li>
        <li>opcion3</li>
    </ul> 
    <button>ACCION</button>    
</body>

Función $().prependTo()

Método equivalente a $().appendTo() pero que añade el nuevo elemento como primer hijo del elemento seleccionado.  El método se llama a partir de la selección del elemento que se quiere añadir, y recibe como argumento la selección del elemento al que quiere añadirse.  

$(<elemento_a_añadir>).prependTo(<elemento_al_que_se_añade>)

Traslado y Clonado de elementos.

El movimiento de elementos consiste en cambiar la posición de elementos ya existentes en el modelo de documento ( DOM ) de la página.

Esto puede conseguirse empleando los métodos ya vistos anteriormente ( $().append(), $().appendTo(), $().prepend(), $().prependTo() ) para añadir elementos ya existentes a otras ubicaciones de la página en vez de elementos nuevos. Esto provoca su cambio de ubicación.

Ejemplo: La siguiente página muestra una lista de opciones con los días de la semana y dos botones “PONER PRIMERO” y “PONER ULTIMO” que desplazan el elemento seleccionado en la lista a la primera o última posición en función del botón pulsado:  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('#btn_primero').click(function() {
                // Desplaza la opción seleccionada a la primera posición 
                $('#dias').prepend($('#dias option:selected')); 
            });
            $('#btn_ultimo').click(function() {
                // Desplaza la opcion seleccionad a la última posición 
                $('#dias').append($('#dias option:selected')); 
            });
        });
    </script>
</head>
<body>

        <option value="3">Miercoles</option>
        <option value="4">Jueves</option>
        <option value="5">Viernes</option>

Clonado de elementos


Si se necesita duplicar un elemento en vez de reubicarlo en la página, es preciso emplear antes el método $().clone(). Este método devuelve una selección con copias de los elementos presentes en la selección a partir de la que se llama. Su sintaxis es la siguiente: 

$<selección_copia> = $(<selección_original>).clone()

Ejemplo: La siguiente página muestra una lista de opciones con frutas vendidas donde el usuario puede ir seleccionando cada fruta y añadirla a la lista de frutas adquiridos pulsando el botón "ADQUIRIR". Las frutas adquiridas deben permanecer no obstante en la lista de frutas vendidas, por lo que  debe clonarse cada opción seleccionada antes de agregarla a la lista de artículos adquiridos: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('#btn_adquirir').click(function() {
                // Obtencion de la opcion seleccionada en lista de frutas vendidas
                var $articulos_seleccionados = $('#articulos option:selected');
                // Agregado del clon de la fruta seleccionada a la lista de frutas adquiridas. 
                $('#compras').append($articulos_seleccionados.clone());

Eliminación de elementos.

JQuery también dispone de funciones que permite eliminar objetos presentes en el modelo de documento ( DOM ). Esto permite eliminar dichos elementos tanto de la vista del usuario como su eliminación del propio modelo. Debe subrayase no obstante, que estos métodos sólo afectan a la página en el navegador del usuario, no a su contenido en el servidor. 

Función $().remove()

Esta función permite eliminar uno o vatios elementos seleccionados  ocasionando que desaparezcan. El método admite dos formas de uso:

  • Sin argumentos : Elimina todos los elementos de la selección contra la que se invoca la función:  

$(<selección_elementos_eliminar>).remove()

  • Con argumentos : Elimina aquellos elementos contenidos en la selección contra la que se invoca la función que cumplen la selección indicada como argumento:  

$(<selección_elementos>).remove(<selección_elementos_contenidos_a_eliminar>)

Ejemplo: La siguiente página modifica el ejemplo de clonado de elementos visto anteriormente añadiendo los botones "ELIMINAR" y "VACIAR": 


El botón “ELIMINAR” elimina la fruta seleccionada de la lista de artículos adquiridos y el botón “VACIAR” elimina todas las frutas dejando la lista vacía.

            $('#btn_eliminar').click(function() {
                // Elimina opciones seleccionadas dentro de lista de opciones 'compras'
                $('#compras option').remove(':selected');

Función $().detach()

Esta función alternativa elimina de la página los elementos seleccionados; pero NO los elimina de la memoria. A diferencia de la función $().remove() ésta  retorna los elementos eliminados para poder volverlos a añadir después al modelo de documento (DOM) empleando funciones vistas con anterioridad ( $().append(), $().appendTo(), $().prepend(), $().prependTo() ). 

Su sintaxis es la siguiente: 

<elementos_eliminados> = <selección_elementos>.detach()

Ejemplo: La siguiente página muestra dos capas con un párrafo y un botón “PULSAME”. 


Cuando se pulsa el botón el párrafo “MENSAJE” es eliminado de la capa superior quedando vacía. 


Al pulsar por segunda vez el botón “PULSAME”, el mismo párrafo es añadido nuevamente a la página en el interior de la segunda capa:


Para eliminar el párrafo contenido en la primera capa se emplea el método $().detach() guardándolo en una variable para poderlo añadir nuevamente después a la segunda capa:

<html>
<head>
    <title></title>
    <style type="text/css">
        .azul {
            width: 100%; 
            height: 100px; 
            background-color: cyan;

            background-color: yellow; 
        }
    </style>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            var $parrafo = null; 
            $("button").click(function () {
                // Esta el párrafo en memoria? 
                if ( $parrafo == null) {
                    // Elimina el párrafo de la primera capa, pero guardándolo en la variable $parrafo.
                    $parrafo = $('div.azul p').detach(); 
                } else {
                    // Reinserta el parrafo en la segunda capa.

Inserción de elementos.

Cuando se añade un elemento al modelo de documento (DOM) mediante las funciones $().append(), $().appendTo(), $().prepend() y $().prependTo(),  éste se añade el primero o el último entre los elementos hijos del seleccionado. Con la inserción sin embargo, un elemento puede agregarse antes o después del elemento seleccionado en su mismo nivel. 

JQuery dispone para ello de las siguientes funciones: 

Función $().after()

Este método inserta el elemento indicado como argumento después del elemento seleccionado como referencia. Su sintaxis es la siguiente: 

<elemento_referencia>.after(<elemento_que_se_inserta>)

Ejemplo: La siguiente página muestra una lista <ol> de elementos: 


tal que al hacer clic sobre una opción queremos que se remarque en amarillo y se añada otra opción “---- nueva opción ----“ justo a continuación empleando para ello el método $().after().  De modo que si hacemos clic en la opción número 3, el resultado es el siguiente:


El código sería el siguiente: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <style type="text/css">
        li { cursor: pointer; }
        .seleccionado { background-color: yellow; }
    </style>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('li').click(function() {
                // Marco el elemento seleccionado
                $(this).addClass('seleccionado');
                // Inserto el nuevo elemento a continuación del seleccionado. 
                $(this).after($('<li>', {text:'--- nueva opcion ---'})); 
            });
        });
    </script>
</head>

<body>    
    <ol>
        <li>opcion</li>
        <li>opcion</li>

        <li>opcion</li>

    </ol>

$().insertAfter()

Este método es equivalente al anterior pero se invoca a partir del elemento a insertar pasando como argumento la selección del elemento junto al que se inserta:  

            <elemento_que_se_inserta>. insertAfter(<elemento_referencia>)

Ejemplo: La siguiente página es equivalente a la del ejemplo anterior pero empleando la función $().insertAfter() en vez de $().after():

Función $().before()


Esta función equivale a $().after(), pero inserta el elemento indicado antes del elemento seleccionado como referencia. Su sintaxis es la siguiente: 

            <elemento_referencia>.before(<elemento_que_se_inserta>)

Ejemplo: Este es el mismo código que el del ejemplo del método $().after() , pero empleando el método $().before().  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <style type="text/css">
        li { cursor: pointer; 
        }
        .seleccionado {
            background-color: yellow; 
        }
    </style>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('li').click(function() {
                // Marco el elemento seleccionado
                $(this).addClass('seleccionado');
                // Inserto el nuevo elemento antes
                $(this).before($('<li>', {text:'--- nueva opcion ---'})); 
            });
        });
    </script>
</head>
<body>    
    <ol>
        <li>opcion</li>
        <li>opcion</li>
        <li>opcion</li>
        <li>opcion</li>
        <li>opcion</li>
        <li>opcion</li>        
    </ol>
</body>
</html>

Sin embargo, al hacer clic sobre cada opción, el nuevo elemento "---nueva opción---" es insertado justo antes debido al uso de $().before():

 

Función $().insertBefore()


Esta función es equivalente a $().before() pero se invoca a partir del elemento a insertar pasando como argumento la selección del elemento junto al que se inserta. La sintaxis es la siguiente:  

<elemento_que_se_inserta>. insertBefore(<elemento_referencia >)

El mismo código del ejemplo anterior podría reescribirse del siguiente modo: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <style type="text/css">
        li { cursor: pointer; 
        }
        .seleccionado {
            background-color: yellow; 
        }
    </style>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('li').click(function() {
                // Marco el elemento seleccionado
                $(this).addClass('seleccionado');
                // Inserto el nuevo elemento antes 
                $('<li>', {text:'--- nueva opcion ---'}).insertBefore($(this)); 
            });
        });
    </script>
</head>
<body>    
    <ol>
        <li>opcion</li>
        <li>opcion</li>
        <li>opcion</li>
        <li>opcion</li>
        <li>opcion</li>
        <li>opcion</li>        
    </ol>
</body>
</html>
