3.1. Captura de eventos.

Eventos y Funciones manejadoras

Javascript permite hacer las páginas web interactivas para que reaccionen a las acciones del usuario sobre sus elementos, pero para ello; es necesario manejar dos conceptos: eventos y manejadores.

  • Un evento es una acción sobre un elemento de la página que puede ser detectada por Javascript ayudándose de JQuery. 
  • Un manejador es una función que se asocia a un evento para que se ejecute cada vez que éste se produzca. 

El manejo de eventos consiste en asociar funciones manejadoras a eventos sobre ciertos elementos de una página web, de modo que al producirse estos eventos, se ejecutan las funciones desempeñando el comportamiento de la página. Esta operación se denomina también  registro de eventos.

JQuery define una función para cada uno de estos eventos que permite asignar a su vez una función manejadora que se ejecute al producirse el evento. Todas estas funciones se llaman a partir de la selección del elemento o elementos cuyo evento quiere manejarse indicando como argumento la declaración de la función manejadora:  

$(<selección_elemento>).evento(<function_manejadora>)

La función más común para registrar eventos es $().click(). Esta función registra el evento de pulsación del botón izquierdo del ratón sobre el elemento o elementos seleccionados asignando como función manejadora la indicada como argumento. 

Ejemplo: El siguiente código muestra una web con un botón “PULSAME” que al ser pulsado modifica el color de fondo de la página al completo. Para ello se declara primero la funcion ‘manejadora’ que se encarga de cambiar color del fondo de la página, y se indica como argumento del método click(): 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Funcion manejadora
            function manejadora(event) {
                $('body').css({'background-color':'blue'});                
            }
            // Registro del evento de pulsacion al boton id='btn_azul'
            $('#btn_azul').click(manejadora);
        });
    </script>
</head>
<body>
      <div>
            <button id='btn_azul'>AZUL</button>
      </div>
</body>
</html>

 
Si la función manejadora sólo va a ser empleada en un único evento es mejor emplear entonces una funcion anónima declarándola directamente sin nombre como argumento de la función:  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Registro del evento de pulsacion al boton id='btn_azul'
            $('#btn_azul').click(function(event) {
                $('body').css({'background-color':'blue'});                
            });
        });
    </script>
</head>
<body>
      <div>
            <button id='btn_azul'>AZUL</button>
      </div>
</body>
</html>

 


Registro de eventos en elementos creados dinámicamente

JQuery no sólo permite registrar eventos asociados a elementos presentes en modelo de documento (DOM) cuando se carga la página en el navegador. También permite registrar eventos asociados a elementos añadidos dinámicamente. La sintaxis es la misma, pero partiendo de elemento recién creados. 

Ejemplo: El siguiente código muestra dos botones “OPCION AZUL” y “OPCION ROJO” que al ser pulsados añaden nuevos elementos <li> a la lista <ul> registrando en cada caso el evento de ‘click’ con una función manejadora distinta: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <style>
        li {
            cursor: pointer;
        }
        .rojo {
            color: red;
        }
        .azul {
            color: blue;
        }
    </style>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('#btn_opcion_rojo')
                .click(function(event) {
                $('ul')
                    .append(
                        // Creación y agregado de nuevo elemento <li> con evento click registrado.
                        $('<li>',{'text':'OPCION ROJO', 'class':'rojo'})
                            .click(function(event) {
                                console.log("ROJO"); 
                            }
                        )
                    );
                });
            $('#btn_opcion_azul')
                .click(function(event) {
                $('ul')
                    .append(
                        // Creación y agregado de nuevo elemento <li> con evento click registrado.
                        $('<li>', {'text':'OPCION AZUL', 'class':'azul'})
                            .on('click', function(event) {
                                console.log("AZUL"); 
                            }
                        )
                    )
                });
        });
    </script>
</head>
<body>
    <ul>
    </ul>
    <button id='btn_opcion_azul'>OPCION AZUL</button>
    <button id='btn_opcion_rojo'>OPCION ROJO</button>
</body>
</html>

Obtención del elemento origen del evento.

Las funciones manejadoras pueden obtener el elemento en el que se originó el evento. Esto resulta útil cuando se emplea una misma función manejadora en distintos elementos. Todas las funciones manejadoras obtienen una referencia al elemento origen del evento mediante la variable predefinida this. Para manejar este elemento mediante los métodos de JQuery debemos tomarlo en una selección. $(this). 

Ejemplo: El siguiente código muestra tres botones “ROJO”, “VERDE” y “AZUL” que al ser pulsados cada uno de ellos llaman una misma función manejadora que muestra en la consola de desarrollador el valor de su atributo id: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Captura del evento click de todos los botones. 
            $('button').click(function(event) {
                // Visualización del atributo ID del botón pulsado en la consola de desarrollador 
                console.log($(this).attr('id')); 
            });
        });
    </script>
</head>
<body>
    <button id='btn_azul'>AZUL</button>
    <button id='btn_rojo'>ROJO</button>
    <button id='btn_verde'>VERDE</button>
</body>
</html>

Ejemplo 2: El siguiente código modifica el anterior haciendo que el color de fondo de la página cambie según el botón que sea pulsado. Para ello, se asigna a cada botón un atributo personalizado ‘ data-color’ con el código hexadecimal correspondiente al color del botón. 

    <button id='btn_azul' data-color='#0000ff'>AZUL</button>
    <button id='btn_rojo' data-color='#ff0000'>ROJO</button>
    <button id='btn_verde' data-color='#00ff00'>VERDE</button>

Desde la función manejadora puede recuperarse el valor del atributo ‘data-color’ del botón pulsado empleando el valor asociado para cambiar el color de fondo de la página. Para ello se emplea la selección $(this) y la función $().data() para obtener el valor del atributo personalizado:  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('button').click(function(event) {
                // Obtencion del valor del atributo data-color del botón pulsado
                var color = $(this).data('color');
                // Asignación del color al fondo de la página
                $('body').css({'background-color':color}); 
            });
        });
    </script>
</head>
<body>
    <button id='btn_azul' data-color='#0000ff'>AZUL</button>
    <button id='btn_rojo' data-color='#ff0000'>ROJO</button>
    <button id='btn_verde' data-color='#00ff00'>VERDE</button>
</body>
</html>

Supresión y Registro múltiple de eventos.

Supresión de Eventos

La función $().off() permite suprimir el evento indicado como argumento de los elementos seleccionados. Su sintaxis es la siguiente:

$(<selección>).off("<evento")

De este modo, se deshabilita la ejecución de las función manejadora que estuviese vinculada al evento suprimido.

Ejemplo: La siguiente página muestra una casilla de verificación y un botón “ACCION”. Cuando la casilla es marcada se registra el evento ‘click’ al botón para que muestre un cuadro de diálogo al ser pulsado. El mismo evento es suprimido al desmarcar la casilla de modo que el botón deja de funcionar: 


<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Evento de cambio de estado de la casilla de verificacion
            $('input:checkbox').change(function(event) {
                if ( $(this).prop('checked')) {
                    // Evento registrado
                    $('button').click(function(event) {
                        alert("BOTON PULSADO!");
                    }); 
                }
                else
                { 
                    // Evento desregistrado
                    $('button').off('click'); 
                }
            });
        });
    </script>
</head>
<body>
    <input type="checkbox">
    EVENTO ACTIVO: 
    <button>ACCION</button>
</body>
</html>


Registro múltiple de eventos


JQuery permite vincular una misma función manejadora a múltiples eventos de los elementos de una selección empleando la función $().on(). Para ello puede emplearse la siguiente sintaxis:  

$(<seleccion>).on("evento1 evento2 ....", function() {...})

Ejemplo: El siguiente código muestra la vinculación de una función manejadora a los eventos mouseenter y mouseleave de todas las capas presentes en la página web:  

            // Registro del evento mouseenter y mouseleave para todas las capas. 
            $("div").on("mouseenter mouseleave", function () {
                …
            });


De igual modo, también puede emplearse la función $().on() para vincular múltiples funciones manejadora a diferentes eventos de los elementos de una selección empleando una única línea de código. Para ello se emplea la siguiente sintaxis:  

$("seleccion").on({
      evento1: function() {...}, 
      evento2: function() {...}, 
      ...
      })
