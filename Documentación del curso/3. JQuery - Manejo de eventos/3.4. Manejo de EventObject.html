3.4. Manejo de EventObject.

El objeto Event

En JQuery las funciones manejadoras de cualquier evento pueden reciben un argumento opcional que es  el objeto event. Este objeto dispone de las propiedades necesarias para obtener información sobre el evento según su tipo:

$( <seleccion > ). evento (function ( event ) { 

         .....
})

Información sobre el evento ( type )


El objeto event dispone de una propiedad type que devuelve una cadena con el identificador del evento sucedido. Esta propiedad puede utilizarse para manejar distintos eventos mediante una misma función manejadora ya que permite identificar el evento que ejecutó la función. 

Ejemplo: El siguiente código muestra una página con una imagen que cambia cuando el puntero del ratón entra y sale de su área registrando los eventos “mouseenter” y “mouseleave” respectivamente y vinculándoles la misma función manejadora a ambos: 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            // Registro de ambos eventos
            $("#imagen").on("mouseenter mouseleave", 
                function(event) {
                    // Evento de puntero entrante. 
                    if (event.type == "mouseenter") {
                        $(this).prop("src", "images/green.png"); 
                    }
                    // Evento de puntero saliente.
                    if (event.type == "mouseleave") {
                        $(this).prop("src", "images/red.png");
                    }                    
                }); 
        });
    </script>
</head>
<body>
    <img id="imagen" src="images/red.png" />
</body>
</html>


Información sobre tecla pulsada ( which )

El objeto event dispone de una propiedad which que retorna un valor entero que indica la tecla pulsada. Esta es una información asociada a los eventos de teclado:

  • KeyDown --> Se dispara el primero cuando se pulsa una tecla.
  • KeyPress --> Se dispara tras KeyDown cuando se pulsa una tecla asociada a un signo, dígito,  carácter, tabulación (Tab), o retorno (Enter). No se dispara con teclas espaciales: Control, Shift, Alt, Borrar, o las teclas de función ( F1 – F12 ).
  • KeyUp --> Se dispara tras los anteriores cuando se libera la tecla pulsada.

 Ejemplo: La siguiente página que muestra una caja de texto que cambia de color al pulsarse las teclas mientras tiene el foco. 

Para que la caja de texto muestre un color de fondo amarillo al pulsarse una tecla cualquiera se registra el evento keyDown:   

            $('input:text').keydown(function(event) {
                $(this).css({'background-color':'yellow'});
            });


Para que la caja de texto recupere un color de fondo blanco cuando se libera la tecla pulsada. Para ello empleamos el evento KeyUp:

            $('input:text').keyup(function(event) {
                $(this).css({'background-color':'white'});                 
            });

Para que la caja de texto muestre un color de fondo rojo si la tecla pulsada es la barra espaciadora se registra el evento KeyPress. Si el valor de la propiedad which es 32 significa que se ha pulsado la barra espaciadora. Se invoca entonces el método preventDefault() para evitar que se añada el espacio en blanco al contenido de la caja de texto: 

(*) Esto puede emplearse para evitar que el usuario pueda introducir ciertos caracteres en el valor de una caja de texto. 

            $('input:text').keypress(function(event) {
                console.log('KEYPRESS'); 
                if ( event.which == 32) {
                    $(this).css({'background-color':'red'});   
                    event.preventDefault();
                } 
            });


El código del la página quedaría del siguiente modo:

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('input:text')
                .keydown(function(event) {
                    $(this).css({'background-color':'yellow'});
                })
                .keyup(function(event) {
                    $(this).css({'background-color':'white'});                 
                })
                .keypress(function(event) {
                    if ( event.which == 32) {
                        $(this).css({'background-color':'red'});   
                        event.preventDefault();
                    }
                });
        });
    </script>
</head>
<body>
    <input type="text">
</body>
</html>


Información sobre posición del ratón ( pageX, pageY )


El objeto Event dispone también de las propiedades pageX y pageY que informan de la posición del ratón en la página en el momento de producirse un evento. Hay que tener presente que estas propiedades sólo están disponibles en eventos de ratón tales como: (mouseover, mouseenter, mouseleave, mousedown, mouseup, mousemove):

  • mousedown : Evento que indica cuando el usuario hace clic con el ratón sobre un elemento.
  • mouseup : Evento que indica cuando el usuario deja de pulsar el ratón estando situado encima del elemento.
  • mouseenter : Evento que indica cuando el puntero del ratón entra en el área ocupada por un elemento.
  • mouseleave : Evento que indica cuando el puntero del ratón abandona el área ocupada por un elemento.
  • mouseover : Evento que indica cuando el puntero del ratón entra en el área ocupada por un elemento, y por sus elementos hijos.
  • mousemove : Evento que indica cada vez que se mueve el ratón dentro del área de un elemento.

 La principal diferencia entre los eventos mouseover y mouseenter es que mouseenter se produce sólo cuando el ratón entra dentro del área del elemento, mientras que mouseover se produce además tanto al entrar como salir del área de elementos hijos

Ejemplo: Sea la siguiente página que muestra una capa “padre” (amarillo) y una capa “hijo” (azul) anidada, ambas con los eventos “mouseenter”, “mouseleave” y “mouseover” registrados para mostrar mensajes en la consola de desarrollo y ver cuándo se disparan al mover el ratón por encima:

    <script>

Obtención de coordenadas del puntero del ratón

Para obtener las coordenadas del puntero del ratón en el momento del evento pueden emplearse las propiedades pageX y pageY del parámetro event de la función manejadora. Estas son coordenadas tienen como origen el extremo superior izquierdo de la página.

Ejemplo: La siguiente página muestra una capa amarilla en la que al hacer clic con el ratón se muestran las coordenadas del puntero en un párrafo. 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <style>
        #padre {
            background-color: yellow;
            border: 1px solid #000;
            position: absolute;
            top: 100px;
            left: 100px;
            width: 400px; 
            height: 400px;
        }
    </style>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $().ready(function () {
            $('#padre')     
                .mousedown(function(event) {
                    $('p').text( event.pageX + "," + event.pageY );
                });
        });
    </script>
</head>
<body>
    <p></p>
    <div id='padre'>
    </div>
</body>
</html>

 
El resultado mostrado al hacer clic dentro de la capa es el siguiente:


Las coordenadas ofrecidas por las propiedades pageX y pageY son referidas al margen superior izquierdo de la página. Si se desea obtener las coordenadas con respecto a la esquina superior izquierda de la propia capa debemos emplear la función offset(): 

        $().ready(function () {
            $('#padre')     
                .mousedown(function(event) {
                    // Obtencion de coordenadas relativas a la capa origen del evento.
                    var x = event.pageX - $(this).offset().left; 
                    var y = event.pageY - $(this).offset().top; 
                    $('p').text( x + "," + y );
                });
        });
