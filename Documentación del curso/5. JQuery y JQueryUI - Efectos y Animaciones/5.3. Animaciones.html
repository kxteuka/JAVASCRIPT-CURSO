5.3. Animaciones

Animaciones

JQuery permite definir animaciones sobre los atributos CSS para crear transiciones entre su valor actual yun valor final a lo largo de un determinado lapso se tiempo. Para ello se emplea la función $().animate().  

$(selector).animate( estilos, lapso )

El primer argumento (estilos) define uno o más estilos con los valores que deben tener al completarse la animación. Estos valores pueden ser absolutos (opacity: 0.5), o relativos (top: “+=25”). 

El segundo parámetro (lapso) indica el lapso de tiempo que dura la transición. Puede expresarse en milisegundos o mediante las cadenas “fast”, “ normal” y “slow” para indicar un lapso de tiempo rápido, medio o lento respectivamente.

Ejemplo: El siguiente código muestra una página con una imagen y un botón "ANIMAR". que al ser pulsado hace que la imagen se desplace 100 píxeles a la derecha respecto de su posición ( left: “+100px” ), mientras se vuelve transparente ( opacity: 0.0 ); en el transcurso de 1 segundo: 


<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <style>
        .imagen {
            position:absolute; 
            top: 100px;
        }
    </style>    
    <script>
        $().ready(function () {
            // Pulsacion boton "OCULTAR"
            $("#btnOculto").click(function () {
                // ANIMACION
                $(".imagen").animate(
                {
                    left: "+=100px",       // mueve +1000px a derecha. 
                    opacity: 0.0            // opacidad pasa al 0%
                },
                1000 );
            });
        });
    </script>
</head>
<body>
    <img class="imagen" src="images/smiley.png" />
    <input type="button" id="btnOculto" value="ANIMAR" />
</body>
</html>


Ejemplo: El siguiente código muestra una página con una imagen que se sitúa centrada sobre la posición del cursor del ratón cada vez que hace clic en cualquier punto de la página. 


Se emplea para ello la función $().animate() seleccionando la imagen para asignar a sus atributos top y left  las coordenadas del cursor del ratón (propiedades pageX y pageY del objeto Event del evento mousedown ) en el momento de hacer clic sobre la página. A estas coordenadas se les resta la mitad del ancho y alto de la imagen para que aparezca centrada sobre posición del cursor: 

<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="scripts/jquery-3.6.0.min.js"></script>    <style>
        .imagen {
            position:absolute; 
        }
    </style>
    <script>
        $().ready(function () {
            // Obtencion tamaño de la imagen
            var ancho_imagen = $(".imagen").width();
            var alto_imagen = $(".imagen").height();
            // Cuando el usuario hace clic sobre la página
            $(window).mousedown(function (e) {
                // Obtencion de la posición del puntero del ratón
                var x = e.pageX;                
                var y = e.pageY;
                // Transicion a la posicion del puntero del ratón
                $(".imagen").animate(           
                        {
                            "top": y - (alto_imagen/2),  // Cambio posición horizontal 
                            "left": x - (ancho_imagen/2) // Cambio posición vertical
                        },
                    1000);  // Lapso: 1 segundo
                });
            });
    </script>
</head>
<body>
    <img class="imagen" src="images/smiley.png" /><br />
</body>
</html>


El parámetro Easing

Este es un parámetro opcional que permite determinar el modo en que se interpolan los valores a lo largo de la animación en la función $().animate(). Los posibles admitidos por el argumento son los siguientes. 

“lineal” : Representa una interpolación lineal en la que los valores varían de igual modo a lo largo de toda la transición. Esto genera una animación continua. 

“swing” : Representa una interpolación suavizada en la que los valores varían más levemente al principio y fin de la transición. Esto genera una animación más suave al principio y final. 


Representación gráfica de la transición de valores “lineal” y “swing”

El parámetro puede indicarse para cada propiedad a animar para establecer el modo en que se interpolan los valores. La sintaxis es la siguiente:  

$().animate(
{
     "propiedad_animar": [valor_final , interpolador  ], 
     "propiedad_animar": [valor_final2, interpolador2 ]
}, duracion )

Ejemplo: Sea una página que muestra dos imágenes a las que se les aplica una animación de 1 segundo de duración en la que ambas se desplazan 500px del margen izquierdo al mismo tiempo. Sin embargo, la imagen superior emplea una interpolación líneal y la inferior una interpolación suavizada (swing).


El movimiento de la imagen superior animada mediante interpolación lineal, mantiene la misma velocidad desde el principio hasta el final. El movimiento de la imagen inferior con interpolación swing varía desplazándose más suavemente al principio y final: 

<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <style>
        .imagen1 {
            position:absolute;
            top: 50px; 
            left: 0px; 
            width: 50px; height: 50px;
        }
        .imagen2 {
            position:absolute;
            top: 150px; 
            left: 0px; 
            width: 50px; height: 50px;
        }
    </style>
    <script>
        $().ready(function () {
            // Cuando el usuario hace clic sobre la página
            $(window).click(function () {
                // Animacion imagen superior
                $(".imagen1").animate(           
                        {
                            "left": [500, "linear"]  // interpolación lineal
                        },
                    1000);
                // Animacion imagen inferior
                $(".imagen2").animate(           
                        {
                            "left": [500, "swing"]  // interpolacion swing
                        },
                    1000);
                   });
         });
    </script>
</head>
<body>
    <p>LINEAL</p>
    <br/><br/><br/>
    <p>SWING</p>
    <img class="imagen1" src="images/smiley.png" /><br />
    <img class="imagen2" src="images/smiley.png" /><br />
</body>
</html>


La función $().animate() también admite opcionalmente una función como tercer argumento para que se ejecute al terminar la animación.

Ejemplo: El siguiente código muestra una animación en la que se desplaza una imagen 300 píxeles a la derecha durante 1 segundo y muestra al terminar el mensaje “Animación completada!” cada vez que se pulsa el botón "ANIMAR":  


<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
    <script src="scripts/jquery-3.6.0.min.js"></script>
    <style>
        .imagen {
            position:absolute; 
            top: 50px;
        }
    </style>    
    <script>
        $().ready(function () {
            // Pulsacion boton "OCULTAR"
            $("#btnOculto").click(function () {
                // ANIMACION
                $(".imagen").animate(
                {
                    left: "+=300px",       // mueve +1000px a derecha desde posición origen. 
                },
                1000,                       // lapso de 1 segundo.
                function() {
                    alert("Animacion completada!");
                });
            });
        });
    </script>
</head>
<body>
    <img class="imagen" src="images/smiley.png" />
    <input type="button" id="btnOculto" value="ANIMAR" />
</body>
</html>
