5.8. Encadenamiento de transiciones

Encadenamiento de transiciones.

Se denomina encadenamiento de transiciones a la aplicación de una sucesión de transiciones, efectos o animaciones sobre uno o varios elementos. 

La siguiente línea de código ejecuta consecutivamente un efecto de sacudida vertical “bounce” y otro de sacudida horizontal “shake” de 1 segundo de duración cada uno sobre el mismo elemento: 

          // Sucesion de efectos. 
          $("#elemento1").effect("bounce", {}, 1000).effect("shake", {}, 1000);

Al invocar varias funciones sobre un mismo elemento o elemento seleccionados, éstas se ejecutan una detrás de otra en el orden en que se invocan. Por ello, primero se ejecutará el efecto “bounce” durante un segundo, y después el “shake” acto seguido: 


<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title></title>
    <!-- Libreria JQuery comprimida -->
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <!-- Tema 'base' para JQueryUI -->
    <link href="https://code.jquery.com/ui/1.13.1/themes/smoothness/jquery-ui.css" rel="stylesheet" />
    <!-- Libreria JQueryUI comprimida -->
    <script src="https://code.jquery.com/ui/1.13.1/jquery-ui.min.js"></script>
    <script>
        $(function () {
            // Evento de click del botón "EFECTO"
            $("#boton_efecto").on("click", function () {
                // Sucesion de efectos. 
                $("#elemento1").effect("bounce", {}, 1000).effect("shake", {}, 1000);
            });
        });
    </script>
</head>
<body>
    <div>
        <div id="elemento1" class="ui-corner-all">
            Etiam libero neque, luctus a, eleifend nec, semper at, lorem. Sed pede.
        </div>
    </div>
    <button id="boton_efecto">EFECTO</button>
</body>
</html>


Sin embargo, los efectos invocados sobre distintos elementos se ejecutan al mismo tiempo. Las siguientes líneas de código ejecutan un efecto “bounce” de 1 segundo sobre un elemento con id = ‘elemento1’, y un efecto “shake” de 1 segundo sobre un elemento con id = ‘elemento2’:
 

                // Efectos simultaneos
                $("#elemento1").effect("bounce", {}, 1000);
                $("#elemento2").effect("shake", {}, 1000);

En este caso, al ser efectos invocados sobre distintos elementos; ambos se ejecutarán al mismo tiempo aunque una línea de código se ejecute antes que la otra:

Para ejecutar consecutivamente efectos aplicados sobre distintos elementos es necesario incluirlos en funciones de retrollamada que se ejecutan al término de cada efecto para invocar el siguiente: 

Las siguientes líneas de código ejecutan un efecto “bounce” de 1 segundo sobre un elemento con id = ‘elemento1’, de modo que al completarse ejecuta la función de retrollamada indicada como último parámetro, que lanza el efecto “shake” sobre el elemento con id = ‘elemento2’: 

               $("#elemento1").effect("bounce", {}, 1000, function() {
                    $("#elemento2").effect("shake", {}, 1000);
               });

En este caso, primero se ejecuta el efecto “bounce” sobre el primer elemento, y al terminar, se ejecuta la función que lanza el efecto “shake” sobre el segundo elemento:

 

Ejemplo: El siguiente código muestra una página con dos capas y tres botones que muestran los tres casos anteriores: 

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title></title>
    <!-- Libreria JQuery comprimida -->
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <!-- Tema 'base' para JQueryUI -->
    <link href="https://code.jquery.com/ui/1.13.1/themes/smoothness/jquery-ui.css" rel="stylesheet" />
    <!-- Libreria JQueryUI comprimida -->
    <script src="https://code.jquery.com/ui/1.13.1/jquery-ui.min.js"></script>
    <script>
        $(function () {
            // Evento de click del botón "EFECTO"
            $("#boton_efecto").on("click", function () {
                // Encadenamiento de efectos en distintos elementos.
                $("#elemento1").effect("bounce", {}, 1000, function() {
                    $("#elemento2").effect("shake", {}, 1000);
                });
            });
        });
    </script>
</head>
<body>
    <div>
        <div id="elemento1" class="ui-corner-all">
            Etiam libero neque, luctus a, eleifend nec, semper at, lorem. Sed pede.
        </div>
        <div id="elemento2" class="ui-corner-all">
            Etiam libero neque, luctus a, eleifend nec, semper at, lorem. Sed pede.
        </div>
    </div>
    <button id="boton_efecto">EFECTO</button>
</body>
</html>
