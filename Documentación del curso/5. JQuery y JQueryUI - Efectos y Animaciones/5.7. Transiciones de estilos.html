5.7. Transiciones de estilos

Transiciones de éstilos $().addClass(), $().removeClass(), $().toggleClass() y $().animate()

JQuery posee funciones que permiten añadir, eliminar y alternar el estilo de los elementos de la página, tales como $().addClass(), $().removeClass() y $().toggleClass(). Sin embargo, los cambios provocados por estas funciones en el aspecto de los elementos es inmediato. 

JQueryUI añade versiones extendidas de dichas funciones que permiten crear animaciones al cambiar los estilos de un elemento.. La sintaxis de las tres funciones es la misma:  

$().addClass(<clase_estilo>, <duración_transicion>, <función_terminacion>);

$().removeClass(<clase_estilo>, <duración_transicion>, <función_terminacion>);

$().toggleClass(<clase_estilo>, <duración_transicion>, <función_terminacion>);

  • La función $().addClass() añade la clase de estilo indicada en el parámetro <clave_estilo> generando una transición entre el aspecto actual y el aspecto final. La duración de la transición viene expresada en milisegundos en el parámetro <duracion>, y puede indicarse opcionalmente una función de retrollamada como parámetro <función> para ser ejecutada al completarse la transición.
  • La función $().removeClass() genera una transición entre el aspecto actual y el resultante al eliminar la clase de estilo indicada en el parámetro <clase_estilo>. Los parámetros aceptados son los mismos.
  • La función $().toggleClass() añade y retira alternativamente con cada llamada el estilo indicado en el parámetro <clase_estilo> generando una transición con la duración especificada.

En los tres casos, las funciones se invocan a partir de una selección de el/los elementos sobre los que aplicar las transiciones. 

Ejemplo: La siguiente página muestra una capa a la que se le añade y elimina la clase de estilo ( newClass ) cuando se pulsan dos botones “addClass” y “removeClass” empleando las transiciones generadas por las funciones $().addClass() y $().removeClass().

Al pulsarse los botones, se produce una transición entre ambos aspectos de un segundo de duración:


 

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title></title>
    <!-- Libreria JQuery comprimida -->
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <!-- Tema 'base' para JQueryUI -->
    <link href="https://code.jquery.com/ui/1.13.1/themes/smoothness/jquery-ui.css" rel="stylesheet" />
    <!-- Libreria JQueryUI comprimida -->
    <script src="https://code.jquery.com/ui/1.13.1/jquery-ui.min.js"></script>
    <style>
        /* Estilo inicial */
        #effect {
            width: 240px;
            padding: 1em;
            border: 1px solid #000;
            background: #eee;
            color: #333;
        }
        /* Estilo nuevo */
        .newClass {
            text-indent: 40px;
            letter-spacing: .4em;
            width: 410px;
            height: 120px;
            padding: 30px;
            margin: 10px;
            font-size: 1.1em;
        }
    </style>
    <script>
        $(function () {
            // Evento de click en boton "AÑADIR ESTILO"
            $("#button_add").on("click", function () {
                 // Añade estilo 'newClass' con transición de estilo de 1 segundo
                $("#effect").addClass("newClass", 1000);
            });
            // Evento de click en boton "ELIMINAR ESTILO"
            $("#button_remove").on("click", function () {
                 // Suprime estilo 'newClass' con transición de estilo de 1 segundo
                $("#effect").removeClass("newClass", 1000);
            });
        });
    </script>
</head>
<body>
    <div class="toggler">
        <div id="effect" class="ui-corner-all">
            Etiam libero neque, luctus a, eleifend nec, semper at, lorem. Sed pede.
        </div>
    </div>
    <!-- Botíon de agregado de estilo -->
    <button id="button_add" class="ui-state-default ui-corner-all">AÑADIR ESTILO</button>
    <!-- Botíon de agregado de estilo -->
    <button id="button_remove" class="ui-state-default ui-corner-all">ELIMINAR ESTILO</button>
</body>
</html>

 

La función $().animate()

La función $().animate() de JQueryUI permite generar transiciones al modificar los estilos asociados a un elemento. 

La función admite un objeto con los estilos a aplicar <estilos> expresados entre llaves y separados por comas, seguido del tiempo de transición en milisegundos <duración>, y una función opcional para que se ejecute al terminar la transición. 

$().animate(<estilos>, <duración>, <función_retrollamada> )

Ejemplo: El siguiente código muestra un formulario con cuatro cajas de texto que al obtener el foco modifican su fondo a un color azulado mediante una transición de color de un cuarto de segundo de duración ( 250 milisegundos ). Cuando pierden el foco recuperan el colo de fondo blanco mediante una transición de la misma duración: 


<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title></title>
    <!-- Libreria JQuery comprimida -->
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <!-- Tema 'base' para JQueryUI -->
    <link href="https://code.jquery.com/ui/1.13.1/themes/smoothness/jquery-ui.css" rel="stylesheet" />
    <!-- Libreria JQueryUI comprimida -->
    <script src="https://code.jquery.com/ui/1.13.1/jquery-ui.min.js"></script>
    <script>
        $(function () {
            // Evento de pérdida de foco de todas las cajas de texto
            $('input:text').blur(function(event) {
                // Animación al color de fondo blanco en 250 milisegundos
               $(this).animate(
                        {
                          backgroundColor: '#ffffff'
                        }, 250);
            });
            // Evento de obtencion de foco de todas las cajas de texto
            $('input:text').focus(function(event) {
                // Animacion al color de fondo azulado en 250 milisegundos
               $(this).animate(
                        {
                          backgroundColor: '#00ffff'
                        }, 250);
            });
        });
    </script>
</head>
<body>
    <form>
        Campo 1<input type='text'><br/>
        Campo 2<input type='text'><br/>
        Campo 3<input type='text'><br/>
        Campo 4<input type='text'><br/>
    </form>
</body>
</html>

Este mismo ejemplo podríamos implementarlo empleando la función $().toggleClass() contra la caja de texto que gane o pierda el foco ( eventos blur() y focus() ) añadiendo y eliminando respectivamente una clase de estilo que muestre la caja de texto seleccionada: 

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title></title>
    <!-- Libreria JQuery comprimida -->
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <!-- Tema 'base' para JQueryUI -->
    <link href="https://code.jquery.com/ui/1.13.1/themes/smoothness/jquery-ui.css" rel="stylesheet" />
    <!-- Libreria JQueryUI comprimida -->
    <script src="https://code.jquery.com/ui/1.13.1/jquery-ui.min.js"></script>
    <style>
        .marcado {
            background-color: #00ffff;
        }
    </style>
    <script>
        $(function () {
            // Evento de pérdida de foco de todas las cajas de texto
            $('input:text').blur(function(event) {
                // asigna/elimina estilo en transición de 250 milisegundos
               $(this).toggleClass('marcado', 250 );
            });
            // Evento de obtencion de foco de todas las cajas de texto
            $('input:text').focus(function(event) {
                // asigna/elimina estilo en transición de 250 milisegundos
               $(this).toggleClass('marcado', 250);
            });
        });
    </script>
</head>
<body>
    <form>
        Campo 1<input type='text'><br/>
        Campo 2<input type='text'><br/>
        Campo 3<input type='text'><br/>
        Campo 4<input type='text'><br/>
    </form>
</body>
</html>
