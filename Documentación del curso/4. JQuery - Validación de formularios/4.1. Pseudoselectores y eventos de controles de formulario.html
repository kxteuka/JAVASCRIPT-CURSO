4.1. Pseudoselectores y eventos de controles de formulario.

Selectores y eventos básicos de formulario

Una de las tareas principales de Javascript es la validación de formularios. Esta consiste básicamente en comprobar que los valores y opciones marcadas en los diferentes elementos que componen un formulario sean válidos. Un ejemplo clásico de validación consiste en comprobar que una caja de texto en la que el usuario debe introducir su DNI no está vacía en el momento de enviar los datos, en cuyo caso; debe cancelarse el envío de éstos al servidor y mostrar un mensaje de advertencia.

Para poder validar cualquier elemento de un formulario, primeramente debemos poder seleccionarlo. JQuery consta de unos selectores específicos para elementos de formulario: 

  •   :button : Selecciona elementos <button> y con el atributo type=’button’
  •    :checkbox : Selecciona elementos <input> con el atributo type='checkbox'
  •    :checked : Selecciona elementos <input> del tipo checkbox seleccionados
  •    :disabled : Selecciona elementos del formulario que están deshabitados
  •    :enabled : Selecciona elementos del formulario que están habilitados
  •    :file : Selecciona elementos <input> con el atributo type='file'
  •    :image : Selecciona elementos <input> con el atributo type='image'
  •    :input : Selecciona elementos <input>, <textarea> y <select>
  •    :password : Selecciona elementos <input> con el atributo type='password'
  •    :radio : Selecciona elementos <input> con el atributo type='radio'
  •    :reset : Selecciona elementos <input> con el atributo type='reset'
  •    :selected : Selecciona elementos <options> que están seleccionados
  •    :submit : Selecciona elementos <input> con el atributo type='submit'
  •    :text : Selecciona elementos <input> con el atributo type='text'

De igual modo, los eventos que suelen capturarse habitualmente en estos elementos para validar los datos indicados o seleccionados por el usuario son los siguientes: 

  • click: evento que se produce cuando se pincha con el ratón sobre un elemento. Normalmente se utiliza con cualquiera de los tipos de botones que permite definir HTML: <input type="button">, <input type="submit"> y <input type="image">.
  • change: evento asociado al cambio del valor de un elemento de texto: <input type="text"> o <textarea>, o la selección de una opción en una lista desplegable <select>. En las listas el evento se dispara en el momento de la selección de una opción. En las cajas de texto se dispara al perder el foco si su valor ha cambiado.
  • focus: evento que se produce cuando el usuario selecciona un elemento del formulario. Se dice entonces que el elemento tiene el “foco”.
  • blur: evento complementario de onfocus que se produce cuando el elemento pierde el “foco” al seleccionar el usuario otro elemento.


Eventos de manejo de formularios mediante JQuery:


Evento click:

La siguiente página muestra un formulario provisto de tres cajas de texto y un botón que al ser pulsado muestra el sumatorio de los valores indicados en una cuarta caja de texto de solo lectura.


La función de cálculo del sumatorio se ejecuta al producirse el evento “click” sobre el botón con valor “cálculo”. Esta selecciona y recorre las cajas de texto donde se encuentran los valores introducidos por el usuario sumándolos a la variable “ sumatorio”. Su valor es finalmente asignado al valor de la caja de texto con id=”resultado”.

<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script> 
    <script>
        $(function () {
            // Seleccion cajas de texto con valores de entrada
            var $cajas = $("input:text:not(#resultado)");
            // Seleccion caja de texto para valor resultado
            var $resultado = $("#resultado"); 
            // Pulsacion botón de cálculo
            $("input:button").click(function () {
                var sumatorio = 0;
                // Obtencion de valores y calculo de sumatorios
                $cajas.each(function () {
                    sumatorio += parseInt($(this).val());
                });
                $resultado.val(sumatorio); 
            })
        });
    </script>
</head>
<body>
    <form>
        valor1: <input type="text"><br>
        valor2: <input type="text"><br>
        valor3: <input type="text"><br>
        <input type="button" value="calculo">
        resultado: <input type="text" id="resultado" readonly>
   </form>
</body>
</html>



Eventos focus y blur:

Se modifica la página anterior de modo que la caja de texto que posea el foco se muestre con el fondo azulado. Para ello se registran los eventos focus y blur de todas ellas exceptuando la de resultado. Cuando una caja gana el foco (focus), se modifica el color de fondo a azulado. Cuando pierde el foco (blur) se le vuelve a asignar el color de fondo original. 

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script> 
    <script>
        $(function () {
            var $cajas = $("input:text:not(#resultado)");
            var $resultado = $("#resultado"); 
            $cajas.focus(function () {                // Obtiene el foco
                $(this).css("background-color", "#00ffff");
            })
            $cajas.blur(function () {                 // Pierde el foco
               $(this).css("background-color", "#ffffff");
            })
            $("input:button").click(function () {
                var sumatorio = 0;
                $cajas.each(function () {
                    sumatorio += parseInt($(this).val());
                    $resultado.val(sumatorio); 
                })
            })
        });
    </script>
</head>
<body>
    <form>
        valor1: <input type="text"><br>
        valor2: <input type="text"><br>
        valor3: <input type="text"><br>
        <input type="button" value="calculo">
        resultado: <input type="text" id="resultado" readonly>
   </form>
</body>
</html>

Manejo de elementos de formulario

Cajas de texto

Los elementos que representan cajas de texto poseen la propiedad value que permite tanto obtener el valor introducido por el usuario como modificarlo desde javascript. Empleando JQuery puede obtener y modificarse el valor de una caja de texto mediante la función $().val().

Ejemplo: Se modifica el ejemplo anterior de modo que al perder el foco cada una de las cajas de texto ( evento blur ) se valida si el valor contenido es un número válido o no. En caso de ser correcto el fondo de la caja de texto se torna blanca, de lo contrario roja indicando la presencia de un valor no numérico.  

<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script> 
    <script>
        $(function () {
            var $cajas = $("input:text:not(#resultado)");
            var $resultado = $("#resultado"); 
            $cajas.focus(function () {                     // Gana el foco
                $(this).css("background-color", "#00ffff");
            })
            $cajas.blur(function (event) {                 // Pierde el foco
                var valor = $(this).val();
                if (!$.isNumeric(valor)) {     // Es un valor numérico??
                    $(this).css("background-color", "#ff0000");
                } else 
                    $(this).css("background-color", "#ffffff");
            })
            $("input:button").click(function () {
                var sumatorio = 0;
                $cajas.each(function () {
                    sumatorio += parseInt($(this).val());
                })
                if (isNaN(sumatorio)) {
                    alert("Se encontraron valores incorrectos."); 
                    $resultado.val("0"); 
                } else
                    $resultado.val(sumatorio); 

            })
        });
    </script>
</head>
<body>
    <form>
        valor1: <input type="text"><br>
        valor2: <input type="text"><br>
        valor3: <input type="text"><br>
        <input type="button" value="calculo">
        resultado: <input type="text" id="resultado" readonly>
   </form>
</body>
</html>


Botones de opción

Los botones de opción permiten seleccionar una opción entre varias de modo que una y sólo una puede ser activa al mismo tiempo. Para ello se asigna el mismo valor al atributo name de todos los elementos. Con JQuery podemos obtener la opción seleccionada mediante el valor del atributo name común y el selector :checked. Para obtener el valor asociado al atributo value del elemento puede emplearse la función de JQuery $().val(). 

Ejemplo: El siguiente código muestra un formulario en el que el usuario puede seleccionar una fruta mediante tres botones de opción. Cuando el usuario pulsa el botón “OK” la página indica si se ha marcado alguna fruta y su valor:  


<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script> 
    <script>
        $(function () {
            // Registro del evento de "click" para el botón
            $("input:button").click(function () {
                // Obtencion de la opcion seleccionada
                $opcion = $("input[name='frutas']:checked");
                // Comprobacion de seleccion de opcion 
                if ($opcion.length > 0) {
                    // Opcion seleccionada
                    alert("La opcion seleccionada " + $opcion.val()); 
                } else {
                    // Ninguna opcion seleccionada
                    alert("No has seleccionada nada"); 
                }
            })
        })
    </script>
</head>
<body>
    <form>
        <input type="radio" value="manzana" name="frutas">Manzana<br>
        <input type="radio" value="platano" name="frutas">Platano<br>
        <input type="radio" value="naranja" name="frutas">Naranja<br>
        <input id="boton" type="button" value="OK">
   </form>
</body>
</html>


Casillas de verificación

Las casillas de verificación son equivalentes a los botones de opción pero no son mutuamente excluyentes y pueden marcarse unas independientemente de otras. Al margen de esta diferencia poseen las mismas propiedades que los botones de opción para seleccionar las que están marcadas mediante el selector "checked:" y obtener el valor de su atributo value mediante la función de $().val() de JQuery. 

También puede comprobarse si la casilla de verificación está marcada empleando la función $().is() indicando como argumento el selector ":checked". Esta función devuelve un valor lógico cierto si la casilla está marcada o falso en caso contrario. 

Ejemplo: El siguiente código muestra un formulario con una casilla de verificación. Cuando el usuario pulsa el botón “OK” se muestra un mensaje indicando si está marcada o no: 


<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script> 
    <script>
        $(function () {
            $("#boton").click(function () {
                // Seleccion de la casilla. Esta marcada?
                if ($("#visa").is(":checked")) {
                    alert("VISA marcado"); 
                } else {
                    alert("VISA no marcado"); 
                }
            })
        })
    </script>
</head>
<body>
    <form>
        <body>
            <input type="checkbox" id="visa">PAGO VISA<br>
            <input id="boton" type="button" value="OK">
        </body>
   </form>
</body>
</html>


Listas de selección

Las listas de selección muestran una lista de valores entre los que se puede seleccionar uno o varios. Cada una de las opciones mostradas se declara como elementos <option>.

Ejemplo: Sea la siguiente lista: 

    <select id="frutas" size="3">
        <option value="1.00">Manzana</option>
        <option value="1.30">Platano</option>
        <option value="0.95">Naranja</option>
    </select>


Para obtener una matriz con todos los elementos de la lista empleamos el selector: 

#frutas option

Para obtener los elementos seleccionados añadimos el selector “:selected”: 

#frutas option :selected

Para comprobar si se ha seleccionado alguna opción se mira que la cantidad de elementos seleccionados sea superior a 0 mediante length: 

                var $seleccion = $("#frutas option:selected");
                // Comprobacion de elemento seleccionado.
                if ($seleccion.length > 0) {
                    // Se ha seleccionado un elemento
                    alert("El elemento seleccionado es: " + $seleccion.html());
                    alert("El valor del producto seleccionado es: " + $seleccion.val());
                } else {
                    // No se ha seleccionado ningún elemento.
                    alert("No se ha seleccionado ninguna fruta");
                }


De cada elemento de la lista puede obtenerse el valor mostrado al usuario con la función $().html(), y el valor de su propiedad value mediante la función $().val().

Ejemplo: El siguiente código muestra una lista de selección simple con tres posibles frutas a elegir. Cuando se pulsa el botón “ OK” (evento click)  se indica si se ha seleccionado alguna fruta mostrando cuál ha sido y el valor del coste en su atributo ‘value’: 

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>

        $(function () {
            $("#boton").click(function () {
                var $seleccion = $("#frutas option:selected");
                // Comprobacion de elemento seleccionado.
                if ($seleccion.length > 0) {
                    // Se ha seleccionado un elemento
                    alert("El elemento seleccionado es: " + $seleccion.html());
                    alert("El valor del producto seleccionado es: " + $seleccion.val());
                } else {
                    // No se ha seleccionado ningún elemento.
                    alert("No se ha seleccionado ninguna fruta");
                }
            });
        })
    </script>
</head>

        <option value="1.00">Manzana</option>

Lista de selección múltiple

Una lista de selección múltiple añade un atributo multiple que permite la selección de varias opciones al mismo tiempo. En este caso, la selección con :selected devuelve todas las opciones seleccionadas que pueden ser varias. La propiedad length indica el total de opciones seleccionadas. Para obtener el valor del atributo value de todas ellas puede recorrerse la selección empleando la función $().each(): 

                var $seleccion = $("#frutas option:selected");
                $seleccion.each(function () {
                        // Mostrar datos de cada opción seleccionada en la lista
                        alert("El elemento seleccionado es: " + $(this).html());
                        alert("El valor del producto seleccionado es: " + $(this).val());
                })

 

Ejemplo: El siguiente código muestra la misma página del ejemplo anterior permitiendo la selección de varias frutas al mismo tiempo. Cuando el usuario pulsa el botón “OK” se indica el nº de opciones seleccionadas y sus correspondientes valores:   

<!DOCTYPE html>
<html>
<head>
    <title></title>
	<meta charset="utf-8" />
    <script src="scripts/jquery-3.2.1.min.js"></script>
    <script>
        $(function () {
            $("#boton").click(function () {
                var $seleccion = $("#frutas option:selected");
                // Comprobacion de elemento seleccionado.
                if ($seleccion.length > 0) {
                    alert("Se han seleccionado: " + $seleccion.length + " opciones.");
                    // Bucle de recorrido de opciones seleccionadas
                    $seleccion.each(function () {
                        // Se ha seleccionado un elemento

                        alert("El valor del producto seleccionado es: " + $(this).val());
                    })
                } else {
                    // No se ha seleccionado ningún elemento.
                    alert("No se ha seleccionado ninguna fruta");
                }
            });
        })
    </script>
</head>

Agregar  opciones a una lista


JQuery permite eliminar y añadir opciones de una lista de formulario durante la ejecución. Esto puede resultar útil en aquellos casos en los que las opciones a mostrar dependen de una selección previa.

Para añadir opciones a una lista desde JQuery pueden seleccionarse el elemento <select> y agregar nuevos elementos <option> mediante la función $().append() ya vista anteriormente: 

Sea por ejemplo una lista <select> con id = "dias", para añadir las opciones "Lunes", "Martes" y "Miércoles" con JQuery emplearíamos el siguiente código: 

2.- Añadir nuevos elementos <option></option> a la lista mediante la función append(): 

	// Agregado a la lista de dias, cada dia en el idioma seleccionado
	$('#dias').append('<option>Lunes</option>');
	$('#dias').append('<option>Martes</option>');
	$('#dias').append('<option>Miercoles</option>');

 

Ejemplo: Sea una web en la que el usuario puede seleccionar un día de la semana en su correspondiente idioma. Para ello la web consta de una lista con 4 idiomas ( castellano, inglés, euskera, y francés ), y una segunda lista donde deben mostrarse los días de la semana en el idioma seleccionado. La idea es emplear JQuery para detectar la selección de un idioma en la lista de idiomas, y agregar los días de la semana en el idioma seleccionada a la segunda lista eliminando los existentes.  


Para obtener el idioma seleccionado por el usuario se captura el evento change de la lista de idiomas. Este evento se produce cada vez que cambia la opción seleccionada. Se obtiene el índice del idioma seleccionado mediante la función $().val()  del elemento seleccionado mediante  “#idiomas option:selected”

La función RefrescarListaDias() recibe el índice del idioma seleccionado y rellena la lista de días con los nombres de los días recogidos en la matriz de matrices "dias":  

	// Matriz de matrices con los dias por idioma. 
	var dias = new Array();
	dias[0] = ['lunes', 'martes', 'miercoles', 'jueves', 'viernes', 'sabado', 'domingo'];
	dias[1] = ['monday', 'tuesday', 'wednesday', 'thursday', 'friday', 'satuday', 'sunday'];
	dias[2] = ['astelehena', 'asteartea', 'asteazkena', 'osteguna', 'ostirala', 'larunbata', 'igandea'];
	dias[3] = ['lundi', 'mardi', 'mercredi', 'jeudi', 'vendredi', 'samedi', 'dimanche'];
	
	// Funcion que refresca la lista de dias según el indice del idioma seleccionado
	function refrescarListaDias( indx ) {
		// Elimina todas los dias en la lista de dias
                $('#dias option').remove();
		// Recorrido de la submatriz de dias en el idioma correspondiente
		$.each(dias[indx], function(indx, val) {
			// Agregado a la lista de dias, cada dia en el idioma seleccionado
			$('#dias').append('<option>' + val + '</option>');
		});
	}

	dias[1] = ['monday', 'tuesday', 'wednesday', 'thursday', 'friday', 'satuday', 'sunday'];
	dias[2] = ['astelehena', 'asteartea', 'asteazkena', 'osteguna', 'ostirala', 'larunbata', 'igandea'];
	dias[3] = ['lundi', 'mardi', 'mercredi', 'jeudi', 'vendredi', 'samedi', 'dimanche'];

Nota: Para recorrer los valores de la matriz de días estamos empleando la función $.each() de JQuery. Esta función permite obtener uno a uno todos los valores contenidos en la matriz indicada como argumento, y no debe confundirse con la función $().each() que se invoca contra una seleccion para obtener uno a uno todos los elementos contenidos.
